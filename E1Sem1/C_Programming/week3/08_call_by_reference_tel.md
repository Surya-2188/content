### Program 1: ‡∞∞‡±Ü‡∞Ç‡∞°‡±Å ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç‡∞≤‡∞®‡∞ø ‡∞Æ‡∞æ‡∞∞‡±ç‡∞ö‡∞°‡∞æ‡∞®‡∞ø‡∞ï‡∞ø (Swap) ‡∞ö‡±á‡∞∏‡∞ø‡∞® ‡∞™‡±ç‡∞∞‡∞Ø‡∞§‡±ç‡∞®‡∞Ç (‡∞á‡∞¶‡∞ø ‡∞™‡∞®‡∞ø‡∞ö‡±á‡∞Ø‡∞¶‡±Å)


‡∞Æ‡∞®‡∞Ç `swap` ‡∞Ö‡∞®‡±á ‡∞´‡∞Ç‡∞ï‡±ç‡∞∑‡∞®‡±ç ‡∞µ‡∞æ‡∞°‡∞ø `a` ‡∞Æ‡∞∞‡∞ø‡∞Ø‡±Å `b` ‡∞µ‡∞ø‡∞≤‡±Å‡∞µ‡∞≤‡∞®‡±Å ‡∞§‡∞æ‡∞∞‡±Å‡∞Æ‡∞æ‡∞∞‡±Å (interchange) ‡∞ö‡±á‡∞Ø‡∞æ‡∞≤‡∞®‡±Å‡∞ï‡±Å‡∞Ç‡∞ü‡±Å‡∞®‡±ç‡∞®‡∞æ‡∞Ç.

```c
#include <stdio.h>


void swap(int x, int y) {
   int temp;
   temp = x;
   x = y;
   y = temp;
}

   int b = 20;


   swap(a, b);


   printf("a = %d, b = %d\n", a, b);
   return 0;
}
```

##  ‡∞Æ‡±Å‡∞Ç‡∞¶‡±Å‡∞ï‡±Å ‡∞µ‡±Ü‡∞≥‡±ç‡∞≥‡±á ‡∞Æ‡±Å‡∞Ç‡∞¶‡±Å ‡∞ï‡±Ä ‡∞™‡∞æ‡∞Ø‡∞ø‡∞Ç‡∞ü‡±ç (Key Idea)

‡∞Ö‡∞°‡±ç‡∞∞‡∞∏‡±ç ‡∞§‡±Ü‡∞≤‡∞ø‡∞∏‡±ç‡∞§‡±á, ‡∞´‡∞Ç‡∞ï‡±ç‡∞∑‡∞®‡±ç ‡∞µ‡±Ü‡∞®‡∞ï‡±ç‡∞ï‡∞ø ‡∞µ‡∞ö‡±ç‡∞ö‡∞ø ‡∞Ü ‡∞Ö‡∞°‡±ç‡∞∞‡∞∏‡±ç ‡∞≤‡±ã ‡∞â‡∞®‡±ç‡∞® ‡∞µ‡∞æ‡∞≤‡±ç‡∞Ø‡±Ç‡∞®‡∞ø ‡∞Æ‡∞æ‡∞∞‡±ç‡∞ö‡∞ó‡∞≤‡∞¶‡±Å ‡∞ï‡∞¶‡∞æ\! ‡∞¶‡±Ä‡∞®‡±ç‡∞®‡±á **Call by Reference** ‡∞Ö‡∞Ç‡∞ü‡∞æ‡∞∞‡±Å.


### Program 2: Call by Reference ‡∞µ‡∞æ‡∞°‡∞ø ‡∞∏‡±ç‡∞µ‡∞æ‡∞™‡±ç ‡∞ö‡±á‡∞Ø‡∞°‡∞Ç (‡∞á‡∞¶‡∞ø ‡∞™‡∞®‡∞ø‡∞ö‡±á‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø)

```c
   // ‡∞ó‡∞Æ‡∞®‡∞ø‡∞Ç‡∞ö‡∞Ç‡∞°‡∞ø: ‡∞Æ‡∞®‡∞Ç '&' ‡∞µ‡∞æ‡∞°‡∞ø ‡∞Ö‡∞°‡±ç‡∞∞‡∞∏‡±ç ‡∞™‡∞Ç‡∞™‡±Å‡∞§‡±Å‡∞®‡±ç‡∞®‡∞æ‡∞Ç

   printf("a = %d, b = %d\n", a, b);
   return 0;
```

### 7\. ‡∞∏‡±ç‡∞ü‡±Ü‡∞™‡±ç-‡∞¨‡±à-‡∞∏‡±ç‡∞ü‡±Ü‡∞™‡±ç ‡∞µ‡∞ø‡∞µ‡∞∞‡∞£

     * `temp` ‡∞≤‡±ã ‡∞â‡∞®‡±ç‡∞® ‡∞™‡∞æ‡∞§ ‡∞µ‡∞æ‡∞≤‡±ç‡∞Ø‡±Ç‡∞®‡∞ø `b` ‡∞≤‡±ã‡∞ï‡∞ø ‡∞ï‡∞æ‡∞™‡±Ä ‡∞ö‡±á‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø.


‡∞á‡∞™‡±ç‡∞™‡±Å‡∞°‡±Å ‡∞Æ‡∞®‡∞Ç ‡∞°‡±à‡∞∞‡±Ü‡∞ï‡±ç‡∞ü‡±ç ‡∞ó‡∞æ ‡∞í‡∞∞‡∞ø‡∞ú‡∞ø‡∞®‡∞≤‡±ç ‡∞Ö‡∞°‡±ç‡∞∞‡∞∏‡±ç ‡∞≤‡∞§‡±ã ‡∞™‡∞®‡∞ø ‡∞ö‡±á‡∞∏‡∞æ‡∞Ç ‡∞ï‡∞æ‡∞¨‡∞ü‡±ç‡∞ü‡∞ø, `a` ‡∞Æ‡∞∞‡∞ø‡∞Ø‡±Å `b` ‡∞®‡∞ø‡∞ú‡∞Ç‡∞ó‡∞æ‡∞®‡±á ‡∞Æ‡∞æ‡∞∞‡∞ø‡∞™‡±ã‡∞§‡∞æ‡∞Ø‡∞ø.

### 8.2 ‡∞´‡∞Ç‡∞ï‡±ç‡∞∑‡∞®‡±ç ‡∞®‡∞ø ‡∞é‡∞≤‡∞æ ‡∞∞‡∞æ‡∞Ø‡∞æ‡∞≤‡∞ø?

### 8.3 ‡∞ï‡∞æ‡∞≤‡±ç ‡∞ö‡±á‡∞∏‡±á‡∞ü‡∞™‡±ç‡∞™‡±Å‡∞°‡±Å ‡∞è‡∞Ç ‡∞™‡∞Ç‡∞™‡∞æ‡∞≤‡∞ø?

**‡∞µ‡∞ø‡∞µ‡∞∞‡∞£:** ‡∞´‡∞Ç‡∞ï‡±ç‡∞∑‡∞®‡±ç `int *x` (‡∞™‡∞æ‡∞Ø‡∞ø‡∞Ç‡∞ü‡∞∞‡±ç) ‡∞®‡∞ø ‡∞Ö‡∞°‡±Å‡∞ó‡±Å‡∞§‡±ã‡∞Ç‡∞¶‡∞ø. ‡∞Ö‡∞Ç‡∞ü‡±á ‡∞¶‡∞æ‡∞®‡∞ø‡∞ï‡∞ø ‡∞Ö‡∞°‡±ç‡∞∞‡∞∏‡±ç ‡∞ï‡∞æ‡∞µ‡∞æ‡∞≤‡∞ø. ‡∞Æ‡∞®‡∞Ç `increment(a)` ‡∞Ö‡∞®‡∞ø ‡∞µ‡∞æ‡∞≤‡±ç‡∞Ø‡±Ç‡∞®‡∞ø ‡∞™‡∞Ç‡∞™‡∞æ‡∞Ç. ‡∞á‡∞¶‡∞ø ‡∞§‡∞™‡±ç‡∞™‡±Å. ‡∞ï‡∞∞‡±Ü‡∞ï‡±ç‡∞ü‡±ç ‡∞ó‡∞æ `increment(&a);` ‡∞Ö‡∞®‡∞ø ‡∞∞‡∞æ‡∞Ø‡∞æ‡∞≤‡∞ø.


### Mistake 3 ‚Äì ‡∞´‡∞Ç‡∞ï‡±ç‡∞∑‡∞®‡±ç ‡∞°‡±Ü‡∞´‡∞ø‡∞®‡∞ø‡∞∑‡∞®‡±ç ‡∞≤‡±ã `*` ‡∞Æ‡∞∞‡±ç‡∞ö‡∞ø‡∞™‡±ã‡∞µ‡∞°‡∞Ç


```c

   *x = *x + 1;

   int a = 5;

   return 0;
```

**‡∞µ‡∞ø‡∞µ‡∞∞‡∞£:** `int x` ‡∞Ö‡∞Ç‡∞ü‡±á ‡∞Æ‡∞æ‡∞Æ‡±Ç‡∞≤‡±Å ‡∞µ‡±á‡∞∞‡∞ø‡∞Ø‡∞¨‡±Å‡∞≤‡±ç, ‡∞™‡∞æ‡∞Ø‡∞ø‡∞Ç‡∞ü‡∞∞‡±ç ‡∞ï‡∞æ‡∞¶‡±Å. ‡∞Æ‡∞æ‡∞Æ‡±Ç‡∞≤‡±Å ‡∞µ‡±á‡∞∞‡∞ø‡∞Ø‡∞¨‡±Å‡∞≤‡±ç ‡∞Æ‡±Ä‡∞¶ `*` ‡∞µ‡∞æ‡∞°‡∞ø‡∞§‡±á ‡∞é‡∞∞‡±ç‡∞∞‡∞∞‡±ç ‡∞µ‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø. ‡∞Ö‡∞°‡±ç‡∞∞‡∞∏‡±ç ‡∞§‡±Ä‡∞∏‡±Å‡∞ï‡±ã‡∞µ‡∞æ‡∞≤‡∞Ç‡∞ü‡±á `int *x` ‡∞Ö‡∞®‡∞ø ‡∞∞‡∞æ‡∞Ø‡∞æ‡∞≤‡∞ø.


### Mistake 4 ‚Äì ‡∞´‡∞Ç‡∞ï‡±ç‡∞∑‡∞®‡±ç ‡∞≤‡±ã‡∞™‡∞≤ `*` ‡∞µ‡∞æ‡∞°‡∞ü‡∞Ç ‡∞Æ‡∞∞‡±ç‡∞ö‡∞ø‡∞™‡±ã‡∞µ‡∞°‡∞Ç

```c
   x = 0;   // ‚ùå
```

**‡∞µ‡∞ø‡∞µ‡∞∞‡∞£:** `x` ‡∞≤‡±ã ‡∞Ö‡∞°‡±ç‡∞∞‡∞∏‡±ç ‡∞â‡∞Ç‡∞¶‡∞ø. `x = 0` ‡∞Ö‡∞®‡∞ø ‡∞∞‡∞æ‡∞∏‡±ç‡∞§‡±á, ‡∞Æ‡∞®‡∞Ç ‡∞™‡∞æ‡∞Ø‡∞ø‡∞Ç‡∞ü‡∞∞‡±ç ‡∞®‡∞ø NULL ‡∞ö‡±á‡∞∏‡±ç‡∞§‡±Å‡∞®‡±ç‡∞®‡∞æ‡∞Ç ‡∞§‡∞™‡±ç‡∞™, ‡∞í‡∞∞‡∞ø‡∞ú‡∞ø‡∞®‡∞≤‡±ç ‡∞µ‡∞æ‡∞≤‡±ç‡∞Ø‡±Ç‡∞®‡∞ø ‡∞Æ‡∞æ‡∞∞‡±ç‡∞ö‡∞ü‡±ç‡∞≤‡±á‡∞¶‡±Å. ‡∞í‡∞∞‡∞ø‡∞ú‡∞ø‡∞®‡∞≤‡±ç ‡∞µ‡∞æ‡∞≤‡±ç‡∞Ø‡±Ç ‡∞Æ‡∞æ‡∞∞‡∞æ‡∞≤‡∞Ç‡∞ü‡±á `*x = 0;` ‡∞Ö‡∞®‡∞ø ‡∞∞‡∞æ‡∞Ø‡∞æ‡∞≤‡∞ø.


### Mistake 5 ‚Äì Call by Value ‡∞Æ‡∞∞‡∞ø‡∞Ø‡±Å Reference ‡∞®‡∞ø ‡∞ï‡∞≤‡∞ó‡∞æ‡∞™‡±Å‡∞≤‡∞ó‡∞Ç ‡∞ö‡±á‡∞Ø‡∞°‡∞Ç

**‡∞µ‡∞ø‡∞µ‡∞∞‡∞£:** `swap` ‡∞∞‡±Ü‡∞Ç‡∞°‡±Å ‡∞Ö‡∞°‡±ç‡∞∞‡∞∏‡±ç ‡∞≤‡∞®‡±Å ‡∞Ö‡∞°‡±Å‡∞ó‡±Å‡∞§‡±ã‡∞Ç‡∞¶‡∞ø. `&a` ‡∞Ö‡∞°‡±ç‡∞∞‡∞∏‡±ç‡∞∏‡±á, ‡∞ï‡∞æ‡∞®‡±Ä `b` ‡∞µ‡∞æ‡∞≤‡±ç‡∞Ø‡±Ç. ‡∞∞‡±Ü‡∞Ç‡∞°‡±Ç ‡∞Ö‡∞°‡±ç‡∞∞‡∞∏‡±ç ‡∞≤‡±Å ‡∞Ö‡∞Ø‡∞ø ‡∞â‡∞Ç‡∞°‡∞æ‡∞≤‡∞ø: `swap(&a, &b);`.


### Mistake 6 ‚Äì ‡∞é‡∞∞‡±á‡∞∏‡±ç ‡∞é‡∞≤‡∞æ ‡∞™‡∞®‡∞ø‡∞ö‡±á‡∞∏‡±ç‡∞§‡∞æ‡∞Ø‡±ã ‡∞Ö‡∞∞‡±ç‡∞•‡∞Ç ‡∞ö‡±á‡∞∏‡±Å‡∞ï‡±ã‡∞ï‡∞™‡±ã‡∞µ‡∞°‡∞Ç

**‡∞µ‡∞ø‡∞µ‡∞∞‡∞£:** ‡∞à ‡∞ï‡±ã‡∞°‡±ç ‡∞®‡∞ø‡∞ú‡∞æ‡∞®‡∞ø‡∞ï‡∞ø **‡∞™‡∞®‡∞ø‡∞ö‡±á‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø**, ‡∞ï‡∞æ‡∞®‡±Ä ‡∞∏‡±ç‡∞ü‡±Ç‡∞°‡±Ü‡∞Ç‡∞ü‡±ç‡∞∏‡±ç ‡∞á‡∞¶‡∞ø ‡∞§‡∞™‡±ç‡∞™‡∞®‡∞ø ‡∞Ö‡∞®‡±Å‡∞ï‡±Å‡∞Ç‡∞ü‡∞æ‡∞∞‡±Å. `arr` ‡∞ï‡∞ø `a` ‡∞Ø‡±ä‡∞ï‡±ç‡∞ï ‡∞Ö‡∞°‡±ç‡∞∞‡∞∏‡±ç ‡∞µ‡±Ü‡∞≥‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø. ‡∞é‡∞∞‡±á‡∞∏‡±ç ‡∞Ü‡∞ü‡±ã‡∞Æ‡±á‡∞ü‡∞ø‡∞ï‡±ç ‡∞ó‡∞æ ‡∞∞‡∞ø‡∞´‡∞∞‡±Ü‡∞®‡±ç‡∞∏‡±ç ‡∞≤‡∞æ ‡∞™‡∞®‡∞ø‡∞ö‡±á‡∞∏‡±ç‡∞§‡∞æ‡∞Ø‡∞ø. ‡∞á‡∞ï‡±ç‡∞ï‡∞° `&` ‡∞Ö‡∞µ‡∞∏‡∞∞‡∞Ç ‡∞≤‡±á‡∞¶‡±Å.


### Mistake 7 ‚Äì ‡∞™‡∞æ‡∞Ø‡∞ø‡∞Ç‡∞ü‡∞∞‡±ç‡∞∏‡±ç ‡∞≤‡±á‡∞ï‡±Å‡∞Ç‡∞°‡∞æ ‡∞∞‡±Ü‡∞Ç‡∞°‡±Å ‡∞µ‡∞æ‡∞≤‡±ç‡∞Ø‡±Ç‡∞∏‡±ç ‡∞∞‡∞ø‡∞ü‡∞∞‡±ç‡∞®‡±ç ‡∞ö‡±á‡∞Ø‡∞æ‡∞≤‡∞®‡±Å‡∞ï‡±ã‡∞µ‡∞°‡∞Ç

**‡∞µ‡∞ø‡∞µ‡∞∞‡∞£:** ‡∞í‡∞ï ‡∞´‡∞Ç‡∞ï‡±ç‡∞∑‡∞®‡±ç ‡∞í‡∞ï‡±ç‡∞ï ‡∞µ‡∞æ‡∞≤‡±ç‡∞Ø‡±Ç‡∞®‡∞ø ‡∞Æ‡∞æ‡∞§‡±ç‡∞∞‡∞Æ‡±á `return` ‡∞ö‡±á‡∞Ø‡∞ó‡∞≤‡∞¶‡±Å. `diff` ‡∞Ö‡∞®‡±á ‡∞µ‡∞æ‡∞≤‡±ç‡∞Ø‡±Ç ‡∞≤‡∞æ‡∞∏‡±ç ‡∞Ö‡∞µ‡±Å‡∞§‡±Å‡∞Ç‡∞¶‡∞ø. ‡∞∞‡±Ü‡∞Ç‡∞°‡±Å ‡∞∞‡∞ø‡∞ú‡∞≤‡±ç‡∞ü‡±ç‡∞∏‡±ç ‡∞ï‡∞æ‡∞µ‡∞æ‡∞≤‡∞Ç‡∞ü‡±á ‡∞™‡∞æ‡∞Ø‡∞ø‡∞Ç‡∞ü‡∞∞‡±ç ‡∞™‡∞æ‡∞∞‡∞æ‡∞Æ‡±Ä‡∞ü‡∞∞‡±ç‡∞∏‡±ç ‡∞µ‡∞æ‡∞°‡∞æ‡∞≤‡∞ø.
