## 2\. ‡∞≤‡∞æ‡∞ú‡∞ø‡∞ï‡±ç (‡∞µ‡∞ø‡∞®‡±ç‡∞®‡∞∞‡±ç & ‡∞∞‡∞®‡±ç‡∞®‡∞∞‡∞™‡±ç ‡∞™‡∞¶‡±ç‡∞ß‡∞§‡∞ø)

**‡∞™‡±ç‡∞∞‡∞æ‡∞∏‡±Ü‡∞∏‡±ç:**

2. **‡∞™‡±ã‡∞≤‡±ç‡∞ö‡∞°‡∞Ç:** ‡∞í‡∞ï‡∞µ‡±á‡∞≥ ‡∞∞‡∞®‡±ç‡∞®‡∞∞‡∞™‡±ç ‡∞ó‡∞æ ‡∞Ö‡∞®‡±Å‡∞ï‡±Å‡∞®‡±ç‡∞® ‡∞µ‡±ç‡∞Ø‡∞ï‡±ç‡∞§‡∞ø ‡∞µ‡∞ø‡∞®‡±ç‡∞®‡∞∞‡±ç ‡∞ï‡∞Ç‡∞ü‡±á ‡∞ó‡±ç‡∞∞‡±á‡∞ü‡±ç ‡∞Ö‡∞Ø‡∞ø‡∞§‡±á, ‡∞µ‡±Ü‡∞Ç‡∞ü‡∞®‡±á ‡∞µ‡∞æ‡∞≥‡±ç‡∞≥ ‡∞™‡±ä‡∞ú‡∞ø‡∞∑‡∞®‡±ç‡∞∏‡±ç ‡∞Æ‡∞æ‡∞∞‡±ç‡∞ö‡∞Ç‡∞°‡∞ø (Swap).
3. **‡∞≤‡±Ç‡∞™‡±ç (Loop):** ‡∞á‡∞™‡±ç‡∞™‡±Å‡∞°‡±Å 3‡∞µ ‡∞µ‡±ç‡∞Ø‡∞ï‡±ç‡∞§‡∞ø (`i = 2`) ‡∞®‡±Å‡∞Ç‡∞°‡∞ø ‡∞Æ‡∞ø‡∞ó‡∞§‡∞æ ‡∞Ö‡∞Ç‡∞¶‡∞∞‡∞ø‡∞®‡∞ø ‡∞ö‡±Ü‡∞ï‡±ç ‡∞ö‡±á‡∞Ø‡∞Ç‡∞°‡∞ø.


‡∞™‡±ç‡∞∞‡∞§‡∞ø ‡∞ï‡±ä‡∞§‡±ç‡∞§ ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç (`arr[i]`) ‡∞µ‡∞ö‡±ç‡∞ö‡∞ø‡∞®‡∞™‡±ç‡∞™‡±Å‡∞°‡±Å, ‡∞∞‡±Ü‡∞Ç‡∞°‡±Å ‡∞™‡±ç‡∞∞‡∞∂‡±ç‡∞®‡∞≤‡±Å ‡∞µ‡±á‡∞∏‡±Å‡∞ï‡±ã‡∞Ç‡∞°‡∞ø:


 * **‡∞™‡±ç‡∞∞‡∞∂‡±ç‡∞® A:** ‡∞à ‡∞ï‡±ä‡∞§‡±ç‡∞§ ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç **‡∞µ‡∞ø‡∞®‡±ç‡∞®‡∞∞‡±ç ‡∞ï‡∞Ç‡∞ü‡±á ‡∞™‡±Ü‡∞¶‡±ç‡∞¶‡∞¶‡∞æ?**
     * **‡∞Ö‡∞µ‡±Å‡∞®‡±Å ‡∞Ö‡∞Ø‡∞ø‡∞§‡±á:** ‡∞™‡∞æ‡∞§ ‡∞µ‡∞ø‡∞®‡±ç‡∞®‡∞∞‡±ç ‡∞ï‡∞ø‡∞Ç‡∞¶‡∞ï‡∞ø ‡∞¶‡∞ø‡∞ó‡∞ø ‡∞∞‡∞®‡±ç‡∞®‡∞∞‡∞™‡±ç ‡∞Ö‡∞µ‡±Å‡∞§‡∞æ‡∞°‡±Å, ‡∞à ‡∞ï‡±ä‡∞§‡±ç‡∞§ ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç ‡∞µ‡∞ø‡∞®‡±ç‡∞®‡∞∞‡±ç ‡∞Ö‡∞µ‡±Å‡∞§‡±Å‡∞Ç‡∞¶‡∞ø.
 * **‡∞™‡±ç‡∞∞‡∞∂‡±ç‡∞® B:** ‡∞™‡±ã‡∞®‡±Ä ‡∞à ‡∞ï‡±ä‡∞§‡±ç‡∞§ ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç **‡∞∞‡∞®‡±ç‡∞®‡∞∞‡∞™‡±ç ‡∞ï‡∞Ç‡∞ü‡±á ‡∞™‡±Ü‡∞¶‡±ç‡∞¶‡∞¶‡∞æ** (‡∞ï‡∞æ‡∞®‡±Ä ‡∞µ‡∞ø‡∞®‡±ç‡∞®‡∞∞‡±ç ‡∞ï‡∞Ç‡∞ü‡±á ‡∞ö‡∞ø‡∞®‡±ç‡∞®‡∞¶‡∞æ)?
     * **‡∞Ö‡∞µ‡±Å‡∞®‡±Å ‡∞Ö‡∞Ø‡∞ø‡∞§‡±á:** ‡∞µ‡∞ø‡∞®‡±ç‡∞®‡∞∞‡±ç ‡∞Ö‡∞≤‡∞æ‡∞ó‡±á ‡∞â‡∞Ç‡∞ü‡∞æ‡∞°‡±Å, ‡∞ï‡∞æ‡∞®‡±Ä ‡∞à ‡∞ï‡±ä‡∞§‡±ç‡∞§ ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç ‡∞∞‡∞®‡±ç‡∞®‡∞∞‡∞™‡±ç ‡∞Ö‡∞µ‡±Å‡∞§‡±Å‡∞Ç‡∞¶‡∞ø.

## 3\. ‡∞∏‡∞ø ‡∞™‡±ç‡∞∞‡±ã‡∞ó‡±ç‡∞∞‡∞æ‡∞Æ‡±ç (The C Program)


```c

   // ‡∞Æ‡±ä‡∞¶‡∞ü‡∞ø ‡∞∞‡±Ü‡∞Ç‡∞°‡±Å ‡∞é‡∞≤‡∞ø‡∞Æ‡±Ü‡∞Ç‡∞ü‡±ç‡∞∏‡±ç ‡∞®‡∞ø largest, secondLargest ‡∞Ö‡∞®‡±Å‡∞ï‡±Å‡∞Ç‡∞¶‡∞æ‡∞Ç

       }

       else if (arr[i] > secondLargest && arr[i] != largest) {

       }
   }

   return 0;
```
