## 2\. ‡∞¨‡∞¨‡±Å‡∞≤‡±ç ‡∞∏‡∞æ‡∞∞‡±ç‡∞ü‡±ç ‡∞é‡∞≤‡∞æ ‡∞™‡∞®‡∞ø‡∞ö‡±á‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø? (‡∞ï‡∞æ‡∞®‡±ç‡∞∏‡±Ü‡∞™‡±ç‡∞ü‡±ç)


‡∞¨‡∞¨‡±Å‡∞≤‡±ç ‡∞∏‡∞æ‡∞∞‡±ç‡∞ü‡±ç ‡∞≤‡∞æ‡∞ú‡∞ø‡∞ï‡±ç ‡∞ö‡∞æ‡∞≤‡∞æ ‡∞∏‡∞ø‡∞Ç‡∞™‡±Å‡∞≤‡±ç: **"‡∞™‡∞ï‡±ç‡∞ï ‡∞™‡∞ï‡±ç‡∞ï‡∞® ‡∞â‡∞®‡±ç‡∞® ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç‡∞∏‡±ç ‡∞®‡∞ø ‡∞™‡±ã‡∞≤‡±ç‡∞ö‡∞ø ‡∞ö‡±Ç‡∞°‡±Å. ‡∞í‡∞ï‡∞µ‡±á‡∞≥ ‡∞Ü‡∞∞‡±ç‡∞°‡∞∞‡±ç ‡∞§‡∞™‡±ç‡∞™‡±Å‡∞ó‡∞æ ‡∞â‡∞Ç‡∞ü‡±á, ‡∞µ‡∞æ‡∞ü‡∞ø‡∞®‡∞ø ‡∞Æ‡∞æ‡∞∞‡±ç‡∞ö‡±á‡∞Ø‡±ç (Swap)."**


**‡∞â‡∞¶‡∞æ‡∞π‡∞∞‡∞£‡∞ï‡∞ø ‡∞à ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç‡∞∏‡±ç ‡∞§‡±Ä‡∞∏‡±Å‡∞ï‡±ã‡∞Ç‡∞°‡∞ø:**

**Pass 1 (‡∞Æ‡±ä‡∞¶‡∞ü‡∞ø ‡∞∞‡±å‡∞Ç‡∞°‡±ç):**

     * 5 ‡∞™‡±Ü‡∞¶‡±ç‡∞¶‡∞¶‡∞ø ‡∞ï‡∞¶‡∞æ? ‡∞Ö‡∞Ç‡∞ü‡±á ‡∞Ü‡∞∞‡±ç‡∞°‡∞∞‡±ç ‡∞§‡∞™‡±ç‡∞™‡±Å. ‡∞∏‡±ã, ‡∞Æ‡∞æ‡∞∞‡±ç‡∞ö‡±á‡∞¶‡±ç‡∞¶‡∞æ‡∞Ç.
     * **‡∞∞‡∞ø‡∞ú‡∞≤‡±ç‡∞ü‡±ç:** `3  5  8  4`

     * 5 ‡∞ö‡∞ø‡∞®‡±ç‡∞®‡∞¶‡±á ‡∞ï‡∞¶‡∞æ? ‡∞Ö‡∞Ç‡∞ü‡±á ‡∞Ü‡∞∞‡±ç‡∞°‡∞∞‡±ç ‡∞ï‡∞∞‡±Ü‡∞ï‡±ç‡∞ü‡±ç. ‡∞è‡∞Ç ‡∞ö‡±á‡∞Ø‡±ä‡∞¶‡±ç‡∞¶‡±Å.
     * **‡∞∞‡∞ø‡∞ú‡∞≤‡±ç‡∞ü‡±ç:** `3  5  8  4`
3. ‡∞á‡∞™‡±ç‡∞™‡±Å‡∞°‡±Å **8** ‡∞®‡∞ø **4** ‡∞®‡∞ø ‡∞™‡±ã‡∞≤‡±ç‡∞ö‡±Å‡∞¶‡∞æ‡∞Ç.
     * 8 ‡∞™‡±Ü‡∞¶‡±ç‡∞¶‡∞¶‡∞ø ‡∞ï‡∞¶‡∞æ? ‡∞Æ‡∞æ‡∞∞‡±ç‡∞ö‡±á‡∞¶‡±ç‡∞¶‡∞æ‡∞Ç.
     * **‡∞∞‡∞ø‡∞ú‡∞≤‡±ç‡∞ü‡±ç:** `3  5  4  8`


**‡∞ó‡∞Æ‡∞®‡∞ø‡∞Ç‡∞ö‡∞æ‡∞∞‡∞æ?** ‡∞í‡∞ï‡±ç‡∞ï ‡∞∞‡±å‡∞Ç‡∞°‡±ç ‡∞Ö‡∞Ø‡±ç‡∞Ø‡∞æ‡∞ï, ‡∞Ö‡∞Ç‡∞¶‡∞∞‡∞ø‡∞ï‡∞Ç‡∞ü‡±á ‡∞™‡±Ü‡∞¶‡±ç‡∞¶ ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç (**8**) ‡∞ö‡∞ø‡∞µ‡∞∞‡∞ø ‡∞™‡±ç‡∞≤‡±á‡∞∏‡±ç ‡∞ï‡∞ø ‡∞µ‡∞ö‡±ç‡∞ö‡±á‡∞∏‡∞ø‡∞Ç‡∞¶‡∞ø. ‡∞®‡±Ä‡∞ü‡∞ø‡∞≤‡±ã ‡∞ó‡∞æ‡∞≤‡∞ø ‡∞¨‡±Å‡∞°‡∞ó (Bubble) ‡∞é‡∞≤‡∞æ ‡∞™‡±à‡∞ï‡∞ø ‡∞§‡±á‡∞≤‡±Å‡∞§‡±Å‡∞Ç‡∞¶‡±ã, ‡∞à ‡∞™‡±Ü‡∞¶‡±ç‡∞¶ ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç ‡∞Ö‡∞≤‡∞æ ‡∞ö‡∞ø‡∞µ‡∞∞‡∞ø‡∞ï‡∞ø ‡∞§‡±á‡∞≤‡∞ø‡∞Ç‡∞¶‡∞ø. ‡∞Ö‡∞Ç‡∞¶‡±Å‡∞ï‡±á ‡∞¶‡±Ä‡∞®‡±ç‡∞®‡∞ø **Bubble Sort** ‡∞Ö‡∞Ç‡∞ü‡∞æ‡∞∞‡±Å.

## 3\. ‡∞∏‡∞ø ‡∞™‡±ç‡∞∞‡±ã‡∞ó‡±ç‡∞∞‡∞æ‡∞Æ‡±ç (The C Program)


```c
#include <stdio.h>


int main() {
   int n, i, j, temp;


   // 1. ‡∞Ö‡∞∞‡±á ‡∞∏‡±à‡∞ú‡±Å ‡∞Ö‡∞°‡∞ó‡∞°‡∞Ç
   printf("Enter number of elements: ");
   scanf("%d", &n);


   int arr[n]; // ‡∞∏‡±à‡∞ú‡±Å ‡∞®‡∞ø ‡∞¨‡∞ü‡±ç‡∞ü‡∞ø ‡∞Ö‡∞∞‡±á ‡∞ï‡±ç‡∞∞‡∞ø‡∞Ø‡±á‡∞ü‡±ç ‡∞ö‡±á‡∞Ø‡∞°‡∞Ç


   // 2. ‡∞é‡∞≤‡∞ø‡∞Æ‡±Ü‡∞Ç‡∞ü‡±ç‡∞∏‡±ç ‡∞á‡∞®‡±ç‡∞™‡±Å‡∞ü‡±ç ‡∞§‡±Ä‡∞∏‡±Å‡∞ï‡±ã‡∞µ‡∞°‡∞Ç
   printf("Enter %d elements:\n", n);
   for(i = 0; i < n; i++) {
       scanf("%d", &arr[i]);
   }


   // 3. ‡∞¨‡∞¨‡±Å‡∞≤‡±ç ‡∞∏‡∞æ‡∞∞‡±ç‡∞ü‡±ç ‡∞≤‡∞æ‡∞ú‡∞ø‡∞ï‡±ç (Bubble Sort Logic)
   // ‡∞¨‡∞Ø‡∞ü‡∞ø ‡∞≤‡±Ç‡∞™‡±ç: ‡∞é‡∞®‡±ç‡∞®‡∞ø ‡∞∞‡±å‡∞Ç‡∞°‡±ç‡∞≤‡±Å (Passes) ‡∞ú‡∞∞‡∞ó‡∞æ‡∞≤‡±ã ‡∞ö‡±Ç‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø
   for(i = 0; i < n - 1; i++) {         
       // ‡∞≤‡±ã‡∞™‡∞≤‡∞ø ‡∞≤‡±Ç‡∞™‡±ç: ‡∞™‡∞ï‡±ç‡∞ï ‡∞™‡∞ï‡±ç‡∞ï‡∞® ‡∞é‡∞≤‡∞ø‡∞Æ‡±Ü‡∞Ç‡∞ü‡±ç‡∞∏‡±ç ‡∞®‡∞ø ‡∞™‡±ã‡∞≤‡±Å‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø
       for(j = 0; j < n - i - 1; j++) { 
           // ‡∞é‡∞°‡∞Æ ‡∞µ‡±à‡∞™‡±Å ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç ‡∞™‡±Ü‡∞¶‡±ç‡∞¶‡∞ó‡∞æ ‡∞â‡∞Ç‡∞ü‡±á, ‡∞Æ‡∞æ‡∞∞‡±ç‡∞ö‡±á‡∞Ø‡±ç (Swap)
           if(arr[j] > arr[j + 1]) {    
               temp = arr[j];
               arr[j] = arr[j + 1];
               arr[j + 1] = temp;
           }
       }
   }


   // 4. ‡∞∏‡∞æ‡∞∞‡±ç‡∞ü‡±ç ‡∞ö‡±á‡∞∏‡∞ø‡∞® ‡∞Ö‡∞∞‡±á ‡∞®‡∞ø ‡∞™‡±ç‡∞∞‡∞ø‡∞Ç‡∞ü‡±ç ‡∞ö‡±á‡∞Ø‡∞°‡∞Ç
   printf("\nSorted Array in Ascending Order:\n");
   for(i = 0; i < n; i++) {
       printf("%d ", arr[i]);
   }
   printf("\n");


   return 0;
}
```
