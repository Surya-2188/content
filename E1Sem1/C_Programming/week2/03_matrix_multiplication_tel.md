‡∞ï‡∞æ‡∞®‡±Ä, ‡∞Æ‡±Ä‡∞∞‡±Å ‡∞í‡∞ï ‡∞ï‡±ç‡∞≤‡∞æ‡∞∏‡±ç ‡∞ü‡±Ä‡∞ö‡∞∞‡±ç ‡∞Ö‡∞®‡±Å‡∞ï‡±ã‡∞Ç‡∞°‡∞ø. **60 ‡∞Æ‡∞Ç‡∞¶‡∞ø ‡∞∏‡±ç‡∞ü‡±Ç‡∞°‡±Ü‡∞Ç‡∞ü‡±ç‡∞∏‡±ç** ‡∞ï‡∞ø, **30 ‡∞∞‡±ã‡∞ú‡±Å‡∞≤** ‡∞Ö‡∞ü‡±Ü‡∞Ç‡∞°‡±Ü‡∞®‡±ç‡∞∏‡±ç (Attendance) ‡∞Æ‡±Ü‡∞Ø‡∞ø‡∞Ç‡∞ü‡±à‡∞®‡±ç ‡∞ö‡±á‡∞Ø‡∞æ‡∞≤‡∞ø.


‡∞í‡∞ï‡∞µ‡±á‡∞≥ ‡∞Æ‡±Ä‡∞∞‡±Å ‡∞Æ‡∞æ‡∞Æ‡±Ç‡∞≤‡±Å ‡∞Ö‡∞∞‡±á ‡∞µ‡∞æ‡∞°‡∞ø‡∞§‡±á:

‡∞á‡∞¶‡∞ø ‡∞ï‡±á‡∞µ‡∞≤‡∞Ç **‡∞í‡∞ï‡±ç‡∞ï ‡∞∞‡±ã‡∞ú‡±Å** ‡∞Ö‡∞ü‡±Ü‡∞Ç‡∞°‡±Ü‡∞®‡±ç‡∞∏‡±ç ‡∞Æ‡∞æ‡∞§‡±ç‡∞∞‡∞Æ‡±á ‡∞∏‡±ç‡∞ü‡±ã‡∞∞‡±ç ‡∞ö‡±á‡∞Ø‡∞ó‡∞≤‡∞¶‡±Å. 30 ‡∞∞‡±ã‡∞ú‡±Å‡∞≤‡∞ï‡±Å 30 ‡∞Ö‡∞∞‡±á‡∞≤‡±Å ‡∞∞‡∞æ‡∞Ø‡∞°‡∞Ç ‡∞ö‡∞æ‡∞≤‡∞æ ‡∞ï‡∞∑‡±ç‡∞ü‡∞Ç.


**‡∞Æ‡∞®‡∞ï‡±Å ‡∞ï‡∞æ‡∞µ‡∞æ‡∞≤‡±ç‡∞∏‡∞ø‡∞Ç‡∞¶‡∞ø ‡∞í‡∞ï ‡∞ü‡±á‡∞¨‡±Å‡∞≤‡±ç (Table):**

 * ‡∞®‡∞ø‡∞≤‡±Å‡∞µ‡±Å‡∞ó‡∞æ (Columns) **‡∞∞‡±ã‡∞ú‡±Å‡∞≤‡±Å**.


‡∞á‡∞≤‡∞æ ‡∞ü‡±á‡∞¨‡±Å‡∞≤‡±ç ‡∞≤‡∞æ‡∞Ç‡∞ü‡∞ø ‡∞°‡±á‡∞ü‡∞æ ‡∞ï‡±ã‡∞∏‡∞Æ‡±á **‡∞Æ‡∞≤‡±ç‡∞ü‡±Ä-‡∞°‡±à‡∞Æ‡±Ü‡∞®‡±ç‡∞∑‡∞®‡∞≤‡±ç ‡∞Ö‡∞∞‡±á (2D Array)** ‡∞ï‡∞®‡∞ø‡∞™‡±Ü‡∞ü‡±ç‡∞ü‡∞æ‡∞∞‡±Å.

## 2\. 2D Array (‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç) ‡∞™‡∞∞‡∞ø‡∞ö‡∞Ø‡∞Ç


2D Array ‡∞Ö‡∞Ç‡∞ü‡±á ‡∞∏‡∞ø‡∞Ç‡∞™‡±Å‡∞≤‡±ç ‡∞ó‡∞æ **‡∞Ö‡∞°‡±ç‡∞°‡±Å ‡∞µ‡∞∞‡±Å‡∞∏‡∞≤‡±Å (Rows)** ‡∞Æ‡∞∞‡∞ø‡∞Ø‡±Å **‡∞®‡∞ø‡∞≤‡±Å‡∞µ‡±Å ‡∞µ‡∞∞‡±Å‡∞∏‡∞≤‡±Å (Columns)** ‡∞â‡∞®‡±ç‡∞® ‡∞í‡∞ï ‡∞¨‡∞æ‡∞ï‡±ç‡∞∏‡±ç.


**‡∞∏‡∞ø‡∞Ç‡∞ü‡∞æ‡∞ï‡±ç‡∞∏‡±ç (Syntax):**


```c

```


**‡∞â‡∞¶‡∞æ‡∞π‡∞∞‡∞£:**

 * `60`: 60 ‡∞Æ‡∞Ç‡∞¶‡∞ø ‡∞∏‡±ç‡∞ü‡±Ç‡∞°‡±Ü‡∞Ç‡∞ü‡±ç‡∞∏‡±ç (Rows).
 * `30`: 30 ‡∞∞‡±ã‡∞ú‡±Å‡∞≤‡±Å (Columns).
 * `attendance[i][j]`: **i-‡∞µ** ‡∞∏‡±ç‡∞ü‡±Ç‡∞°‡±Ü‡∞Ç‡∞ü‡±ç, **j-‡∞µ** ‡∞∞‡±ã‡∞ú‡±Å ‡∞µ‡∞ö‡±ç‡∞ö‡∞æ‡∞°‡∞æ ‡∞≤‡±á‡∞¶‡∞æ ‡∞Ö‡∞®‡∞ø ‡∞ö‡±Ü‡∞™‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø.

## 3\. ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞Æ‡∞≤‡±ç‡∞ü‡∞ø‡∞™‡±ç‡∞≤‡∞ø‡∞ï‡±á‡∞∑‡∞®‡±ç 


‡∞∞‡±Ü‡∞Ç‡∞°‡±Å ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞≤‡∞®‡±Å ‡∞ï‡±Ç‡∞°‡∞ü‡∞Ç (Addition) ‡∞ö‡∞æ‡∞≤‡∞æ ‡∞à‡∞ú‡±Ä (‡∞®‡±á‡∞∞‡±Å‡∞ó‡∞æ ‡∞ï‡±Ç‡∞°‡∞ü‡∞Æ‡±á). ‡∞ï‡∞æ‡∞®‡±Ä **‡∞ó‡±Å‡∞£‡∞ï‡∞æ‡∞∞‡∞Ç (Multiplication)** ‡∞Ö‡∞≤‡∞æ ‡∞ï‡∞æ‡∞¶‡±Å. ‡∞á‡∞¶‡∞ø ‡∞ï‡±ä‡∞Ç‡∞ö‡±Ü‡∞Ç ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞ï‡±Ä‡∞ó‡∞æ ‡∞â‡∞Ç‡∞ü‡±Å‡∞Ç‡∞¶‡∞ø.


### ‡∞¨‡∞Ç‡∞ó‡∞æ‡∞∞‡±Å ‡∞∏‡±Ç‡∞§‡±ç‡∞∞‡∞æ‡∞≤‡±Å (Golden Rules)


1. **‡∞ï‡∞Ç‡∞°‡∞ø‡∞∑‡∞®‡±ç:** ‡∞∞‡±Ü‡∞Ç‡∞°‡±Å ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞≤‡∞®‡±Å ‡∞ó‡±Å‡∞£‡∞ø‡∞Ç‡∞ö‡∞æ‡∞≤‡∞Ç‡∞ü‡±á...
     * ‡∞Æ‡±ä‡∞¶‡∞ü‡∞ø ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞≤‡±ã ‡∞â‡∞®‡±ç‡∞® **Columns**, ‡∞∞‡±Ü‡∞Ç‡∞°‡±ã ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞≤‡±ã ‡∞â‡∞®‡±ç‡∞® **Rows** ‡∞ï‡∞ø ‡∞∏‡∞Æ‡∞æ‡∞®‡∞Ç‡∞ó‡∞æ ‡∞â‡∞Ç‡∞°‡∞æ‡∞≤‡∞ø.
       $$Columns \ of \ A == Rows \ of \ B$$
2. **‡∞∏‡±à‡∞ú‡±Å:** ‡∞í‡∞ï‡∞µ‡±á‡∞≥ A ‡∞∏‡±à‡∞ú‡±Å $(m \times n)$ ‡∞Æ‡∞∞‡∞ø‡∞Ø‡±Å B ‡∞∏‡±à‡∞ú‡±Å $(n \times p)$ ‡∞Ö‡∞Ø‡∞ø‡∞§‡±á, ‡∞µ‡∞ö‡±ç‡∞ö‡±á ‡∞Ü‡∞®‡±ç‡∞∏‡∞∞‡±ç C ‡∞∏‡±à‡∞ú‡±Å $(m \times p)$ ‡∞Ö‡∞µ‡±Å‡∞§‡±Å‡∞Ç‡∞¶‡∞ø.


### ‡∞´‡∞æ‡∞∞‡±ç‡∞Æ‡±Å‡∞≤‡∞æ


‡∞í‡∞ï‡±ç‡∞ï ‡∞¨‡∞æ‡∞ï‡±ç‡∞∏‡±ç ‡∞®‡∞ø‡∞Ç‡∞™‡∞°‡∞æ‡∞®‡∞ø‡∞ï‡∞ø, ‡∞Æ‡∞®‡∞Ç **‡∞Æ‡±ä‡∞¶‡∞ü‡∞ø ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞≤‡±ã‡∞®‡∞ø ‡∞Ö‡∞°‡±ç‡∞°‡±Å ‡∞µ‡∞∞‡±Å‡∞∏ (Row)** ‡∞®‡∞ø, **‡∞∞‡±Ü‡∞Ç‡∞°‡±ã ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞≤‡±ã‡∞®‡∞ø ‡∞®‡∞ø‡∞≤‡±Å‡∞µ‡±Å ‡∞µ‡∞∞‡±Å‡∞∏ (Col)** ‡∞§‡±ã ‡∞ó‡±Å‡∞£‡∞ø‡∞Ç‡∞ö‡∞ø ‡∞ï‡±Ç‡∞°‡∞æ‡∞≤‡∞ø.

## 4\. "3 ‡∞≤‡±Ç‡∞™‡±ç‡∞∏‡±ç ‡∞é‡∞Ç‡∞¶‡±Å‡∞ï‡±Å ‡∞ï‡∞æ‡∞µ‡∞æ‡∞≤‡∞ø?" (‡∞Ö‡∞Ç‡∞¶‡∞∞‡±Ç ‡∞Ö‡∞°‡∞ø‡∞ó‡±á ‡∞°‡±å‡∞ü‡±ç)


‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞Ö‡∞°‡∞ø‡∞∑‡∞®‡±ç ‡∞ï‡∞ø 2 ‡∞≤‡±Ç‡∞™‡±ç‡∞∏‡±ç ‡∞∏‡∞∞‡∞ø‡∞™‡±ã‡∞§‡∞æ‡∞Ø‡∞ø, ‡∞Æ‡∞∞‡∞ø ‡∞Æ‡∞≤‡±ç‡∞ü‡∞ø‡∞™‡±ç‡∞≤‡∞ø‡∞ï‡±á‡∞∑‡∞®‡±ç ‡∞ï‡∞ø **3 ‡∞≤‡±Ç‡∞™‡±ç‡∞∏‡±ç (`i`, `j`, `k`)** ‡∞é‡∞Ç‡∞¶‡±Å‡∞ï‡±Å?


**‡∞≤‡∞æ‡∞ú‡∞ø‡∞ï‡±ç:**


1. **Loop 1 (`i`):** ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç A ‡∞≤‡±ã **Row** ‡∞®‡∞ø ‡∞∏‡±Ü‡∞≤‡±Ü‡∞ï‡±ç‡∞ü‡±ç ‡∞ö‡±á‡∞∏‡±Å‡∞ï‡±ã‡∞µ‡∞°‡∞æ‡∞®‡∞ø‡∞ï‡∞ø.
2. **Loop 2 (`j`):** ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç B ‡∞≤‡±ã **Column** ‡∞®‡∞ø ‡∞∏‡±Ü‡∞≤‡±Ü‡∞ï‡±ç‡∞ü‡±ç ‡∞ö‡±á‡∞∏‡±Å‡∞ï‡±ã‡∞µ‡∞°‡∞æ‡∞®‡∞ø‡∞ï‡∞ø.
3. **Loop 3 (`k`):** ‡∞á‡∞¶‡∞ø **‡∞ï‡∞æ‡∞≤‡∞ø‡∞ï‡±ç‡∞Ø‡±Å‡∞≤‡±á‡∞ü‡∞∞‡±ç ‡∞≤‡±Ç‡∞™‡±ç**.


**`k` ‡∞≤‡±Ç‡∞™‡±ç ‡∞é‡∞Ç‡∞¶‡±Å‡∞ï‡±Å?**
‡∞Ü‡∞®‡±ç‡∞∏‡∞∞‡±ç ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞≤‡±ã ‡∞í‡∞ï‡±ç‡∞ï ‡∞¨‡∞æ‡∞ï‡±ç‡∞∏‡±ç (`C[i][j]`) ‡∞®‡∞ø‡∞Ç‡∞™‡∞æ‡∞≤‡∞Ç‡∞ü‡±á ‡∞Æ‡∞®‡∞Ç ‡∞ö‡∞æ‡∞≤‡∞æ ‡∞™‡∞®‡±Å‡∞≤‡±Å ‡∞ö‡±á‡∞Ø‡∞æ‡∞≤‡∞ø:


 * Row A ‡∞≤‡±ã ‡∞Æ‡±ä‡∞¶‡∞ü‡∞ø ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç $\times$ Col B ‡∞≤‡±ã ‡∞Æ‡±ä‡∞¶‡∞ü‡∞ø ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç.
 * Row A ‡∞≤‡±ã ‡∞∞‡±Ü‡∞Ç‡∞°‡±ã ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç $\times$ Col B ‡∞≤‡±ã ‡∞∞‡±Ü‡∞Ç‡∞°‡±ã ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç.
 * ... ‡∞á‡∞≤‡∞æ ‡∞µ‡∞∞‡±Å‡∞∏‡∞ó‡∞æ ‡∞ó‡±Å‡∞£‡∞ø‡∞Ç‡∞ö‡∞ø, ‡∞µ‡∞æ‡∞ü‡∞®‡±ç‡∞®‡∞ø‡∞Ç‡∞ü‡∞ø‡∞®‡±Ä **‡∞ï‡±Ç‡∞°‡∞æ‡∞≤‡∞ø (Add)**.


‡∞à ‡∞™‡∞®‡∞ø‡∞®‡∞ø ‡∞∞‡∞ø‡∞™‡±Ä‡∞ü‡±ç ‡∞ö‡±á‡∞Ø‡∞°‡∞æ‡∞®‡∞ø‡∞ï‡∞ø ‡∞Æ‡∞®‡∞ï‡±Å ‡∞Æ‡±Ç‡∞°‡±ã ‡∞≤‡±Ç‡∞™‡±ç (`k`) ‡∞ï‡∞æ‡∞µ‡∞æ‡∞≤‡∞ø.

## 5\. ‡∞™‡±Ç‡∞∞‡±ç‡∞§‡∞ø ‡∞∏‡∞ø ‡∞™‡±ç‡∞∞‡±ã‡∞ó‡±ç‡∞∞‡∞æ‡∞Æ‡±ç (The Complete C Program)


```c
#include <stdio.h>


int main() {
   int r1, c1, r2, c2;
   int i, j, k;


   // 1. Matrix A ‡∞∏‡±à‡∞ú‡±Å ‡∞Ö‡∞°‡∞ó‡∞°‡∞Ç
   printf("Enter rows and columns of Matrix A: ");
   scanf("%d %d", &r1, &c1);


   // 2. Matrix B ‡∞∏‡±à‡∞ú‡±Å ‡∞Ö‡∞°‡∞ó‡∞°‡∞Ç
   printf("Enter rows and columns of Matrix B: ");
   scanf("%d %d", &r2, &c2);


   // 3. ‡∞ï‡∞Ç‡∞°‡∞ø‡∞∑‡∞®‡±ç ‡∞ö‡±Ü‡∞ï‡±ç ‡∞ö‡±á‡∞Ø‡∞°‡∞Ç (‡∞Ö‡∞§‡∞ø ‡∞Æ‡±Å‡∞ñ‡±ç‡∞Ø‡∞Æ‡±à‡∞®‡∞¶‡∞ø)
   if (c1 != r2) {
       printf("Error! Multiplication not possible.\n");
       printf("Columns of A (%d) must match Rows of B (%d).\n", c1, r2);
       return 0;
   }


   // 4. ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞°‡∞ø‡∞ï‡±ç‡∞≤‡±á‡∞∞‡±ç ‡∞ö‡±á‡∞Ø‡∞°‡∞Ç
   int A[r1][c1], B[r2][c2], C[r1][c2];


   // 5. Matrix A ‡∞á‡∞®‡±ç‡∞™‡±Å‡∞ü‡±ç
   printf("\nEnter elements of Matrix A:\n");
   for (i = 0; i < r1; i++) {
       for (j = 0; j < c1; j++) {
           scanf("%d", &A[i][j]);
       }
   }


   // 6. Matrix B ‡∞á‡∞®‡±ç‡∞™‡±Å‡∞ü‡±ç
   printf("\nEnter elements of Matrix B:\n");
   for (i = 0; i < r2; i++) {
       for (j = 0; j < c2; j++) {
           scanf("%d", &B[i][j]);
       }
   }


   // 7. ‡∞Ü‡∞®‡±ç‡∞∏‡∞∞‡±ç ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç C ‡∞®‡∞ø 0 ‡∞§‡±ã ‡∞®‡∞ø‡∞Ç‡∞™‡∞°‡∞Ç
   // ‡∞á‡∞¶‡∞ø ‡∞ö‡±á‡∞Ø‡∞ï‡∞™‡±ã‡∞§‡±á ‡∞™‡∞æ‡∞§ ‡∞ó‡∞æ‡∞∞‡±ç‡∞¨‡±á‡∞ú‡±ç ‡∞µ‡∞æ‡∞≤‡±ç‡∞Ø‡±Ç‡∞∏‡±ç ‡∞Ø‡∞æ‡∞°‡±ç ‡∞Ö‡∞Ø‡∞ø ‡∞Ü‡∞®‡±ç‡∞∏‡∞∞‡±ç ‡∞§‡∞™‡±ç‡∞™‡±Å ‡∞µ‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø
   for (i = 0; i < r1; i++) {
       for (j = 0; j < c2; j++) {
           C[i][j] = 0;
       }
   }


   // 8. ‡∞Ö‡∞∏‡∞≤‡±à‡∞® ‡∞≤‡∞æ‡∞ú‡∞ø‡∞ï‡±ç (3 Nested Loops)
   for (i = 0; i < r1; i++) {           // A ‡∞≤‡±ã‡∞®‡∞ø Rows ‡∞ï‡±ã‡∞∏‡∞Ç
       for (j = 0; j < c2; j++) {       // B ‡∞≤‡±ã‡∞®‡∞ø Columns ‡∞ï‡±ã‡∞∏‡∞Ç
           for (k = 0; k < c1; k++) {   // ‡∞ó‡±Å‡∞£‡∞ø‡∞Ç‡∞ö‡∞ø ‡∞ï‡±Ç‡∞°‡∞ü‡∞Ç ‡∞ï‡±ã‡∞∏‡∞Ç
               C[i][j] += A[i][k] * B[k][j];
           }
       }
   }


   // 9. ‡∞Ü‡∞®‡±ç‡∞∏‡∞∞‡±ç ‡∞™‡±ç‡∞∞‡∞ø‡∞Ç‡∞ü‡±ç ‡∞ö‡±á‡∞Ø‡∞°‡∞Ç
   printf("\nResultant Matrix C (A x B):\n");
   for (i = 0; i < r1; i++) {
       for (j = 0; j < c2; j++) {
           printf("%d ", C[i][j]);
       }
       printf("\n"); // ‡∞ü‡±á‡∞¨‡±Å‡∞≤‡±ç ‡∞´‡∞æ‡∞∞‡±ç‡∞Æ‡∞æ‡∞ü‡±ç ‡∞ï‡±ã‡∞∏‡∞Ç
   }


   return 0;
}
```
