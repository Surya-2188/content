## 1\. ‡∞Ö‡∞∏‡∞≤‡±Å ‡∞ü‡±ç‡∞∞‡∞æ‡∞®‡±ç‡∞∏‡±ç‚Äå‡∞™‡±ã‡∞ú‡±ç ‡∞Ö‡∞Ç‡∞ü‡±á ‡∞è‡∞Ç‡∞ü‡∞ø?


‡∞ü‡±ç‡∞∞‡∞æ‡∞®‡±ç‡∞∏‡±ç‚Äå‡∞™‡±ã‡∞ú‡±ç ‡∞Ö‡∞Ç‡∞ü‡±á ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞®‡∞ø ‡∞Ö‡∞°‡±ç‡∞°‡∞Ç‡∞ó‡∞æ ‡∞§‡∞ø‡∞™‡±ç‡∞™‡∞°‡∞Ç. ‡∞∏‡∞ø‡∞Ç‡∞™‡±Å‡∞≤‡±ç ‡∞ó‡∞æ ‡∞ö‡±Ü‡∞™‡±ç‡∞™‡∞æ‡∞≤‡∞Ç‡∞ü‡±á:


 * **‡∞Ö‡∞°‡±ç‡∞°‡±Å ‡∞µ‡∞∞‡±Å‡∞∏‡∞≤‡±Å (Rows)** ‡∞Ö‡∞®‡±ç‡∞®‡±Ä **‡∞®‡∞ø‡∞≤‡±Å‡∞µ‡±Å ‡∞µ‡∞∞‡±Å‡∞∏‡∞≤‡±Å (Columns)** ‡∞Ö‡∞µ‡±Å‡∞§‡∞æ‡∞Ø‡∞ø.
 * **‡∞®‡∞ø‡∞≤‡±Å‡∞µ‡±Å ‡∞µ‡∞∞‡±Å‡∞∏‡∞≤‡±Å (Columns)** ‡∞Ö‡∞®‡±ç‡∞®‡±Ä **‡∞Ö‡∞°‡±ç‡∞°‡±Å ‡∞µ‡∞∞‡±Å‡∞∏‡∞≤‡±Å (Rows)** ‡∞Ö‡∞µ‡±Å‡∞§‡∞æ‡∞Ø‡∞ø.


**‡∞â‡∞¶‡∞æ‡∞π‡∞∞‡∞£:**
Matrix **A** ($2 \times 3$) ‡∞á‡∞≤‡∞æ ‡∞â‡∞Ç‡∞ü‡±á:



‡∞¶‡∞æ‡∞®‡∞ø ‡∞ü‡±ç‡∞∞‡∞æ‡∞®‡±ç‡∞∏‡±ç‚Äå‡∞™‡±ã‡∞ú‡±ç **T** ($3 \times 2$) ‡∞á‡∞≤‡∞æ ‡∞Æ‡∞æ‡∞∞‡±Å‡∞§‡±Å‡∞Ç‡∞¶‡∞ø:



‡∞ö‡±Ç‡∞∏‡∞æ‡∞∞‡∞æ? ‡∞Ö‡∞°‡±ç‡∞°‡∞Ç‡∞ó‡∞æ ‡∞â‡∞®‡±ç‡∞® `1 2 3` ‡∞ï‡∞æ‡∞∏‡±ç‡∞§‡∞æ ‡∞®‡∞ø‡∞≤‡±Å‡∞µ‡±Å‡∞ó‡∞æ ‡∞Æ‡∞æ‡∞∞‡∞ø‡∞Ç‡∞¶‡∞ø.

## 2\. ‡∞≤‡∞æ‡∞ú‡∞ø‡∞ï‡±ç: ‡∞á‡∞Ç‡∞°‡±Ü‡∞ï‡±ç‡∞∏‡±ç ‡∞Æ‡∞æ‡∞∞‡±ç‡∞ö‡∞°‡∞Ç (Swapping Indices)


‡∞á‡∞¶‡±á ‡∞Ö‡∞∏‡∞≤‡±à‡∞® ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç.


 * ‡∞í‡∞∞‡∞ø‡∞ú‡∞ø‡∞®‡∞≤‡±ç ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞≤‡±ã ‡∞í‡∞ï ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç `A[i][j]` (Row `i`, Column `j`) ‡∞¶‡∞ó‡±ç‡∞ó‡∞∞ ‡∞â‡∞Ç‡∞ü‡±á...
 * ‡∞ü‡±ç‡∞∞‡∞æ‡∞®‡±ç‡∞∏‡±ç‚Äå‡∞™‡±ã‡∞ú‡±ç ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞≤‡±ã ‡∞Ü ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç `T[j][i]` (Row `j`, Column `i`) ‡∞¶‡∞ó‡±ç‡∞ó‡∞∞‡∞ø‡∞ï‡∞ø ‡∞µ‡±Ü‡∞≥‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø.


**‡∞´‡∞æ‡∞∞‡±ç‡∞Æ‡±Å‡∞≤‡∞æ:**
$$T[j][i] = A[i][j]$$

## 3\. ‡∞∏‡∞ø ‡∞™‡±ç‡∞∞‡±ã‡∞ó‡±ç‡∞∞‡∞æ‡∞Æ‡±ç (The C Program)


```c
#include <stdio.h>


int main() {
   int rows, cols, i, j;


   // 1. ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞∏‡±à‡∞ú‡±Å ‡∞Ö‡∞°‡∞ó‡∞°‡∞Ç
   printf("Enter number of rows: ");
   scanf("%d", &rows);


   printf("Enter number of columns: ");
   scanf("%d", &cols);


   // 2. ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞°‡∞ø‡∞ï‡±ç‡∞≤‡±á‡∞∞‡±ç ‡∞ö‡±á‡∞Ø‡∞°‡∞Ç
   // ‡∞Æ‡±Å‡∞ñ‡±ç‡∞Ø‡∞Ç: T ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞∏‡±à‡∞ú‡±Å ‡∞∞‡∞ø‡∞µ‡∞∞‡±ç‡∞∏‡±ç ‡∞≤‡±ã ‡∞â‡∞Ç‡∞°‡∞æ‡∞≤‡∞ø!
   int A[rows][cols];
   int T[cols][rows];


   // 3. Matrix A ‡∞á‡∞®‡±ç‡∞™‡±Å‡∞ü‡±ç
   printf("\nEnter elements of Matrix A:\n");
   for(i = 0; i < rows; i++) {
       for(j = 0; j < cols; j++) {
           scanf("%d", &A[i][j]);
       }
   }


   // 4. ‡∞ü‡±ç‡∞∞‡∞æ‡∞®‡±ç‡∞∏‡±ç‚Äå‡∞™‡±ã‡∞ú‡±ç ‡∞ö‡±á‡∞Ø‡∞°‡∞Ç (‡∞Ö‡∞∏‡∞≤‡±à‡∞® ‡∞≤‡∞æ‡∞ú‡∞ø‡∞ï‡±ç)
   for(i = 0; i < rows; i++) {
       for(j = 0; j < cols; j++) {
           // [i][j] ‡∞≤‡±ã ‡∞â‡∞®‡±ç‡∞®‡∞¶‡∞æ‡∞®‡±ç‡∞®‡∞ø [j][i] ‡∞≤‡±ã ‡∞µ‡±Ü‡∞Ø‡±ç‡∞Ø‡∞ø
           T[j][i] = A[i][j];
       }
   }


   // 5. ‡∞ü‡±ç‡∞∞‡∞æ‡∞®‡±ç‡∞∏‡±ç‚Äå‡∞™‡±ã‡∞ú‡±ç ‡∞®‡∞ø ‡∞™‡±ç‡∞∞‡∞ø‡∞Ç‡∞ü‡±ç ‡∞ö‡±á‡∞Ø‡∞°‡∞Ç
   // ‡∞ó‡∞Æ‡∞®‡∞ø‡∞ï: ‡∞á‡∞™‡±ç‡∞™‡±Å‡∞°‡±Å T ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞ï‡∞ø 'cols' ‡∞Ö‡∞®‡±á‡∞µ‡∞ø Rows ‡∞Ö‡∞µ‡±Å‡∞§‡∞æ‡∞Ø‡∞ø
   printf("\nTranspose of Matrix A:\n");
   for(i = 0; i < cols; i++) {
       for(j = 0; j < rows; j++) {
           printf("%d ", T[i][j]);
       }
       printf("\n");
   }


   return 0;
}
```

## 4\. ‡∞≤‡±à‡∞®‡±ç-‡∞¨‡±à-‡∞≤‡±à‡∞®‡±ç ‡∞µ‡∞ø‡∞µ‡∞∞‡∞£


### ‡∞∏‡±ç‡∞ü‡±Ü‡∞™‡±ç 1: ‡∞°‡∞ø‡∞ï‡±ç‡∞≤‡∞∞‡±á‡∞∑‡∞®‡±ç (The Flip)

‡∞á‡∞¶‡∞ø ‡∞Ö‡∞§‡±ç‡∞Ø‡∞Ç‡∞§ ‡∞Æ‡±Å‡∞ñ‡±ç‡∞Ø‡∞Æ‡±à‡∞® ‡∞∏‡±ç‡∞ü‡±Ü‡∞™‡±ç.
‡∞í‡∞ï‡∞µ‡±á‡∞≥ Matrix A ‡∞∏‡±à‡∞ú‡±Å **2 Rows, 3 Columns** ($2 \times 3$) ‡∞Ö‡∞Ø‡∞ø‡∞§‡±á...

‡∞∏‡±à‡∞ú‡±Å‡∞≤‡±Å ‡∞§‡∞æ‡∞∞‡±Å‡∞Æ‡∞æ‡∞∞‡±Å ‡∞ö‡±á‡∞Ø‡∞ï‡∞™‡±ã‡∞§‡±á, ‡∞°‡±á‡∞ü‡∞æ ‡∞∏‡∞∞‡∞ø‡∞™‡±ã‡∞ï ‡∞™‡±ç‡∞∞‡±ã‡∞ó‡±ç‡∞∞‡∞æ‡∞Æ‡±ç ‡∞ï‡±ç‡∞∞‡∞æ‡∞∑‡±ç ‡∞Ö‡∞µ‡±Å‡∞§‡±Å‡∞Ç‡∞¶‡∞ø.


### ‡∞∏‡±ç‡∞ü‡±Ü‡∞™‡±ç 2: ‡∞≤‡∞æ‡∞ú‡∞ø‡∞ï‡±ç ‡∞≤‡±Ç‡∞™‡±ç

‡∞Æ‡∞®‡∞Ç Matrix A ‡∞≤‡±ã‡∞®‡∞ø ‡∞™‡±ç‡∞∞‡∞§‡∞ø ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç ‡∞¶‡∞ó‡±ç‡∞ó‡∞∞‡∞ø‡∞ï‡∞ø ‡∞µ‡±Ü‡∞≥‡±ç‡∞§‡∞æ‡∞Ç. `[i][j]` ‡∞≤‡±ã ‡∞â‡∞®‡±ç‡∞® ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç ‡∞®‡∞ø ‡∞§‡±Ä‡∞∏‡±Å‡∞ï‡±Å‡∞®‡∞ø, T ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞≤‡±ã ‡∞∞‡∞ø‡∞µ‡∞∞‡±ç‡∞∏‡±ç ‡∞™‡±ä‡∞ú‡∞ø‡∞∑‡∞®‡±ç `[j][i]` ‡∞≤‡±ã ‡∞™‡±Ü‡∞°‡∞§‡∞æ‡∞Ç.


### ‡∞∏‡±ç‡∞ü‡±Ü‡∞™‡±ç 3: ‡∞™‡±ç‡∞∞‡∞ø‡∞Ç‡∞ü‡∞ø‡∞Ç‡∞ó‡±ç

‡∞á‡∞ï‡±ç‡∞ï‡∞° ‡∞≤‡±Ç‡∞™‡±ç `rows` ‡∞§‡±ã ‡∞ï‡∞æ‡∞ï‡±Å‡∞Ç‡∞°‡∞æ `cols` ‡∞§‡±ã ‡∞é‡∞Ç‡∞¶‡±Å‡∞ï‡±Å ‡∞∏‡±ç‡∞ü‡∞æ‡∞∞‡±ç‡∞ü‡±ç ‡∞ö‡±á‡∞∂‡∞æ‡∞Ç?
‡∞é‡∞Ç‡∞¶‡±Å‡∞ï‡∞Ç‡∞ü‡±á ‡∞ü‡±ç‡∞∞‡∞æ‡∞®‡±ç‡∞∏‡±ç‚Äå‡∞™‡±ã‡∞ú‡±ç ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç `T` ‡∞≤‡±ã ‡∞Ö‡∞°‡±ç‡∞°‡±Å ‡∞µ‡∞∞‡±Å‡∞∏‡∞≤ ‡∞∏‡∞Ç‡∞ñ‡±ç‡∞Ø ‡∞í‡∞∞‡∞ø‡∞ú‡∞ø‡∞®‡∞≤‡±ç ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç `cols` ‡∞ï‡∞ø ‡∞∏‡∞Æ‡∞æ‡∞®‡∞Ç ‡∞ï‡∞æ‡∞¨‡∞ü‡±ç‡∞ü‡∞ø.

## üîç 5. ‡∞°‡±ç‡∞∞‡±à ‡∞∞‡∞®‡±ç (Trace: ‡∞Ö‡∞∏‡∞≤‡±Å ‡∞≤‡±ã‡∞™‡∞≤ ‡∞è‡∞Ç ‡∞ú‡∞∞‡±Å‡∞ó‡±Å‡∞§‡±ã‡∞Ç‡∞¶‡∞ø?)

## üõë 6. ‡∞∏‡±ç‡∞ü‡±Ç‡∞°‡±Ü‡∞Ç‡∞ü‡±ç‡∞∏‡±ç ‡∞ö‡±á‡∞∏‡±á ‡∞ï‡∞æ‡∞Æ‡∞®‡±ç ‡∞§‡∞™‡±ç‡∞™‡±Å‡∞≤‡±Å (Common Mistakes)


### ‚ùå ‡∞§‡∞™‡±ç‡∞™‡±Å 1: ‡∞∏‡±à‡∞ú‡±Å‡∞≤‡±Å ‡∞Æ‡∞æ‡∞∞‡±ç‡∞ö‡∞°‡∞Ç ‡∞Æ‡∞∞‡±ç‡∞ö‡∞ø‡∞™‡±ã‡∞µ‡∞°‡∞Ç

### ‚ùå ‡∞§‡∞™‡±ç‡∞™‡±Å 2: ‡∞ü‡±ç‡∞∞‡∞æ‡∞®‡±ç‡∞∏‡±ç‚Äå‡∞™‡±ã‡∞ú‡±ç ‡∞ï‡∞ø ‡∞¨‡∞¶‡±Å‡∞≤‡±Å ‡∞ï‡∞æ‡∞™‡±Ä ‡∞ö‡±á‡∞Ø‡∞°‡∞Ç

 * **‡∞é‡∞Ç‡∞¶‡±Å‡∞ï‡±Å:** ‡∞á‡∞¶‡∞ø ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞®‡∞ø ‡∞§‡∞ø‡∞™‡±ç‡∞™‡∞¶‡±Å, ‡∞â‡∞®‡±ç‡∞®‡∞¶‡∞ø ‡∞â‡∞®‡±ç‡∞®‡∞ü‡±ç‡∞≤‡±Å ‡∞ï‡∞æ‡∞™‡±Ä ‡∞ö‡±á‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø.
 * **‡∞ï‡∞∞‡±Ü‡∞ï‡±ç‡∞ü‡±ç:** `T[j][i] = A[i][j];` (‡∞á‡∞Ç‡∞°‡±Ü‡∞ï‡±ç‡∞∏‡±ç ‡∞≤‡±Å ‡∞Æ‡∞æ‡∞∞‡±ç‡∞ö‡∞æ‡∞≤‡∞ø).


### ‚ùå ‡∞§‡∞™‡±ç‡∞™‡±Å 3: ‡∞™‡±ç‡∞∞‡∞ø‡∞Ç‡∞ü‡∞ø‡∞Ç‡∞ó‡±ç ‡∞≤‡±Ç‡∞™‡±ç‡∞∏‡±ç ‡∞≤‡±ã ‡∞ï‡∞®‡±ç‡∞´‡±ç‡∞Ø‡±Ç‡∞ú‡∞®‡±ç


 * **‡∞§‡∞™‡±ç‡∞™‡±Å:** T ‡∞®‡∞ø ‡∞™‡±ç‡∞∞‡∞ø‡∞Ç‡∞ü‡±ç ‡∞ö‡±á‡∞∏‡±á‡∞ü‡∞™‡±ç‡∞™‡±Å‡∞°‡±Å ‡∞™‡∞æ‡∞§ `rows` ‡∞≤‡∞ø‡∞Æ‡∞ø‡∞ü‡±ç ‡∞µ‡∞æ‡∞°‡∞ü‡∞Ç.
 * **‡∞é‡∞Ç‡∞¶‡±Å‡∞ï‡±Å:** T ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞°‡±à‡∞Æ‡±Ü‡∞®‡±ç‡∞∑‡∞®‡±ç‡∞∏‡±ç ‡∞Æ‡∞æ‡∞∞‡∞ø‡∞™‡±ã‡∞Ø‡∞æ‡∞Ø‡∞ø. T ‡∞Ø‡±ä‡∞ï‡±ç‡∞ï Rows ‡∞∏‡∞Ç‡∞ñ‡±ç‡∞Ø `cols`.
