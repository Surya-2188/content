## 1\. ‡∞ï‡±ç‡∞µ‡∞ø‡∞ï‡±ç ‡∞∞‡±Ä‡∞ï‡∞æ‡∞≤‡±ç: ‡∞Ö‡∞∏‡∞≤‡±Å ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞Ö‡∞Ç‡∞ü‡±á ‡∞è‡∞Ç‡∞ü‡∞ø?


**‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç** ‡∞Ö‡∞Ç‡∞ü‡±á ‡∞∏‡∞ø‡∞Ç‡∞™‡±Å‡∞≤‡±ç ‡∞ó‡∞æ ‡∞í‡∞ï **2D Array**. ‡∞á‡∞¶‡∞ø ‡∞ö‡±Ç‡∞°‡∞ü‡∞æ‡∞®‡∞ø‡∞ï‡∞ø ‡∞Ö‡∞°‡±ç‡∞° ‡∞ó‡±Ä‡∞§‡∞≤‡±Å (Rows) ‡∞Æ‡∞∞‡∞ø‡∞Ø‡±Å ‡∞®‡∞ø‡∞≤‡±Å‡∞µ‡±Å ‡∞ó‡±Ä‡∞§‡∞≤‡±Å (Columns) ‡∞â‡∞®‡±ç‡∞® ‡∞í‡∞ï **‡∞ü‡±á‡∞¨‡±Å‡∞≤‡±ç** ‡∞≤‡∞æ‡∞ó‡∞æ ‡∞â‡∞Ç‡∞ü‡±Å‡∞Ç‡∞¶‡∞ø.

## 2\. "‡∞∞‡±Ü‡∞Ç‡∞°‡±Å ‡∞≤‡±Ç‡∞™‡±ç‡∞∏‡±ç" ‡∞∞‡±Ç‡∞≤‡±ç (‡∞ö‡∞æ‡∞≤‡∞æ ‡∞Æ‡±Å‡∞ñ‡±ç‡∞Ø‡∞Ç\!)


‡∞Æ‡∞®‡∞Ç ‡∞ï‡±ã‡∞°‡±ç ‡∞∞‡∞æ‡∞∏‡±á ‡∞Æ‡±Å‡∞Ç‡∞¶‡±Å, ‡∞í‡∞ï ‡∞∞‡±Ç‡∞≤‡±ç ‡∞ó‡±Å‡∞∞‡±ç‡∞§‡±Å‡∞™‡±Ü‡∞ü‡±ç‡∞ü‡±Å‡∞ï‡±ã‡∞µ‡∞æ‡∞≤‡∞ø:


> **üõë ‡∞∞‡±Ç‡∞≤‡±ç:** 2D Array (‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç) ‡∞§‡±ã ‡∞™‡∞®‡∞ø ‡∞ö‡±á‡∞Ø‡∞æ‡∞≤‡∞Ç‡∞ü‡±á ‡∞Æ‡∞®‡∞ï‡±Å ‡∞ï‡∞ö‡±ç‡∞ö‡∞ø‡∞§‡∞Ç‡∞ó‡∞æ **‡∞∞‡±Ü‡∞Ç‡∞°‡±Å ‡∞≤‡±Ç‡∞™‡±ç‡∞∏‡±ç (Nested Loops)** ‡∞ï‡∞æ‡∞µ‡∞æ‡∞≤‡∞ø.


**‡∞é‡∞Ç‡∞¶‡±Å‡∞ï‡±Å? ‡∞í‡∞ï ‡∞∏‡±ç‡∞ï‡±Ç‡∞≤‡±ç ‡∞¨‡∞ø‡∞≤‡±ç‡∞°‡∞ø‡∞Ç‡∞ó‡±ç ‡∞®‡∞ø ‡∞ä‡∞π‡∞ø‡∞Ç‡∞ö‡±Å‡∞ï‡±ã‡∞Ç‡∞°‡∞ø.**
‡∞Ü ‡∞∏‡±ç‡∞ï‡±Ç‡∞≤‡±ç ‡∞≤‡±ã **‡∞Ö‡∞Ç‡∞§‡∞∏‡±ç‡∞§‡±Å‡∞≤‡±Å (Rows)** ‡∞â‡∞®‡±ç‡∞®‡∞æ‡∞Ø‡∞ø, ‡∞™‡±ç‡∞∞‡∞§‡∞ø ‡∞Ö‡∞Ç‡∞§‡∞∏‡±ç‡∞§‡±Å‡∞≤‡±ã **‡∞ó‡∞¶‡±Å‡∞≤‡±Å (Columns)** ‡∞â‡∞®‡±ç‡∞®‡∞æ‡∞Ø‡∞ø.
‡∞Æ‡±Ä‡∞∞‡±Å ‡∞™‡±ç‡∞∞‡∞§‡∞ø ‡∞ó‡∞¶‡∞ø‡∞®‡∞ø ‡∞ï‡±ç‡∞≤‡±Ä‡∞®‡±ç ‡∞ö‡±á‡∞Ø‡∞æ‡∞≤‡∞Ç‡∞ü‡±á, ‡∞®‡±á‡∞∞‡±Å‡∞ó‡∞æ ‡∞™‡∞∞‡∞ø‡∞ó‡±Ü‡∞§‡±ç‡∞§‡∞≤‡±á‡∞∞‡±Å ‡∞ï‡∞¶‡∞æ?


1. **‡∞¨‡∞Ø‡∞ü‡∞ø ‡∞≤‡±Ç‡∞™‡±ç (‡∞≤‡∞ø‡∞´‡±ç‡∞ü‡±ç ‡∞≤‡∞æ‡∞Ç‡∞ü‡∞ø‡∞¶‡∞ø):** ‡∞Æ‡±Å‡∞Ç‡∞¶‡±Å **Floor 0** ‡∞ï‡∞ø ‡∞µ‡±Ü‡∞≥‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø.
2. **‡∞≤‡±ã‡∞™‡∞≤‡∞ø ‡∞≤‡±Ç‡∞™‡±ç (‡∞®‡∞°‡∞ï ‡∞≤‡∞æ‡∞Ç‡∞ü‡∞ø‡∞¶‡∞ø):** ‡∞Ü ‡∞´‡±ç‡∞≤‡±ã‡∞∞‡±ç ‡∞≤‡±ã ‡∞â‡∞®‡±ç‡∞® **Room 0**, ‡∞§‡∞∞‡±ç‡∞µ‡∞æ‡∞§ **Room 1**, ‡∞§‡∞∞‡±ç‡∞µ‡∞æ‡∞§ **Room 2** ‡∞®‡∞ø ‡∞µ‡∞ø‡∞ú‡∞ø‡∞ü‡±ç ‡∞ö‡±á‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø.
3. **‡∞¨‡∞Ø‡∞ü‡∞ø ‡∞≤‡±Ç‡∞™‡±ç:** ‡∞™‡∞®‡∞ø ‡∞Ö‡∞Ø‡±ç‡∞Ø‡∞æ‡∞ï **Floor 1** ‡∞ï‡∞ø ‡∞§‡±Ä‡∞∏‡±Å‡∞ï‡±Ü‡∞≥‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø.
4. **‡∞≤‡±ã‡∞™‡∞≤‡∞ø ‡∞≤‡±Ç‡∞™‡±ç:** ‡∞Æ‡∞≥‡±ç‡∞≥‡±Ä ‡∞Ö‡∞ï‡±ç‡∞ï‡∞°‡±Å‡∞®‡±ç‡∞® ‡∞ó‡∞¶‡±Å‡∞≤‡∞®‡±ç‡∞®‡∞ø‡∞Ç‡∞ü‡∞ø‡∞®‡±Ä ‡∞µ‡∞ø‡∞ú‡∞ø‡∞ü‡±ç ‡∞ö‡±á‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø.


**‡∞™‡±ç‡∞∞‡±ã‡∞ó‡±ç‡∞∞‡∞æ‡∞Æ‡∞ø‡∞Ç‡∞ó‡±ç ‡∞≠‡∞æ‡∞∑‡∞≤‡±ã:**


 * **Outer Loop (`i`):** ‡∞á‡∞¶‡∞ø **Rows** ‡∞®‡∞ø ‡∞ï‡∞Ç‡∞ü‡±ç‡∞∞‡±ã‡∞≤‡±ç ‡∞ö‡±á‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø (‡∞®‡∞ø‡∞¶‡∞æ‡∞®‡∞Ç‡∞ó‡∞æ ‡∞ï‡∞¶‡±Å‡∞≤‡±Å‡∞§‡±Å‡∞Ç‡∞¶‡∞ø).
 * **Inner Loop (`j`):** ‡∞á‡∞¶‡∞ø **Columns** ‡∞®‡∞ø ‡∞ï‡∞Ç‡∞ü‡±ç‡∞∞‡±ã‡∞≤‡±ç ‡∞ö‡±á‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø (‡∞´‡∞æ‡∞∏‡±ç‡∞ü‡±ç ‡∞ó‡∞æ ‡∞ï‡∞¶‡±Å‡∞≤‡±Å‡∞§‡±Å‡∞Ç‡∞¶‡∞ø).


‡∞∞‡±Ü‡∞Ç‡∞°‡±Å ‡∞≤‡±Ç‡∞™‡±ç‡∞∏‡±ç ‡∞≤‡±á‡∞ï‡∞™‡±ã‡∞§‡±á, ‡∞Æ‡∞®‡∞Ç ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞≤‡±ã‡∞®‡∞ø ‡∞™‡±ç‡∞∞‡∞§‡∞ø ‡∞ó‡∞°‡∞ø‡∞®‡∞ø (cell) ‡∞ü‡∞ö‡±ç ‡∞ö‡±á‡∞Ø‡∞≤‡±á‡∞Ç.

## 3\. ‡∞≤‡∞æ‡∞ú‡∞ø‡∞ï‡±ç: ‡∞Ö‡∞°‡∞ø‡∞∑‡∞®‡±ç ‡∞é‡∞≤‡∞æ ‡∞ö‡±á‡∞Ø‡∞æ‡∞≤‡∞ø?


‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞Ö‡∞°‡∞ø‡∞∑‡∞®‡±ç ‡∞ö‡∞æ‡∞≤‡∞æ ‡∞∏‡∞ø‡∞Ç‡∞™‡±Å‡∞≤‡±ç. ‡∞Æ‡±ä‡∞¶‡∞ü‡∞ø ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞≤‡±ã ‡∞â‡∞®‡±ç‡∞® ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç ‡∞®‡∞ø, ‡∞∞‡±Ü‡∞Ç‡∞°‡±ã ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞≤‡±ã **‡∞Ö‡∞¶‡±á ‡∞™‡±ä‡∞ú‡∞ø‡∞∑‡∞®‡±ç** ‡∞≤‡±ã ‡∞â‡∞®‡±ç‡∞® ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç ‡∞ï‡∞ø ‡∞ï‡±Ç‡∞°‡∞æ‡∞≤‡∞ø.


**‡∞´‡∞æ‡∞∞‡±ç‡∞Æ‡±Å‡∞≤‡∞æ:**
$$C[i][j] = A[i][j] + B[i][j]$$


**‡∞¨‡±ä‡∞Æ‡±ç‡∞Æ ‡∞∞‡±Ç‡∞™‡∞Ç‡∞≤‡±ã:**


```text
   Matrix A         Matrix B          Result (C)
 [ 1   2 ]        [ 4   5 ]        [ 1+4   2+5 ]
 [ 3   4 ]   +    [ 6   7 ]   =    [ 3+6   4+7 ]
```


**‡∞∑‡∞∞‡∞§‡±Å (Condition):**
‡∞∞‡±Ü‡∞Ç‡∞°‡±Å ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞≤‡∞ï‡±Å **‡∞∏‡±à‡∞ú‡±Å ‡∞∏‡∞Æ‡∞æ‡∞®‡∞Ç‡∞ó‡∞æ** (same rows & columns) ‡∞â‡∞Ç‡∞ü‡±á‡∞®‡±á ‡∞Æ‡∞®‡∞Ç ‡∞ï‡±Ç‡∞°‡∞ó‡∞≤‡∞Ç.

## 4\. ‡∞∏‡∞ø ‡∞™‡±ç‡∞∞‡±ã‡∞ó‡±ç‡∞∞‡∞æ‡∞Æ‡±ç (The C Program)


```c
#include <stdio.h>


int main() {
   int rows, cols, i, j;


   // 1. ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞∏‡±à‡∞ú‡±Å ‡∞é‡∞Ç‡∞§‡±ã ‡∞®‡∞ø‡∞∞‡±ç‡∞£‡∞Ø‡∞ø‡∞Ç‡∞ö‡±Å‡∞ï‡±ã‡∞µ‡∞°‡∞Ç
   printf("Enter number of rows: ");
   scanf("%d", &rows);


   printf("Enter number of columns: ");
   scanf("%d", &cols);


   // ‡∞Æ‡±Ç‡∞°‡±Å 2D Arrays (Matrices) ‡∞°‡∞ø‡∞ï‡±ç‡∞≤‡±á‡∞∞‡±ç ‡∞ö‡±á‡∞Ø‡∞°‡∞Ç
   int A[rows][cols], B[rows][cols], C[rows][cols];


   // 2. Matrix A ‡∞á‡∞®‡±ç‡∞™‡±Å‡∞ü‡±ç ‡∞§‡±Ä‡∞∏‡±Å‡∞ï‡±ã‡∞µ‡∞°‡∞Ç (‡∞∞‡±Ü‡∞Ç‡∞°‡±Å ‡∞≤‡±Ç‡∞™‡±ç‡∞∏‡±ç ‡∞µ‡∞æ‡∞°‡∞ø)
   printf("\nEnter elements of Matrix A:\n");
   for(i = 0; i < rows; i++) {           // Outer Loop: Row ‡∞Æ‡∞æ‡∞∞‡±Å‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø
       for(j = 0; j < cols; j++) {       // Inner Loop: Column ‡∞Æ‡∞æ‡∞∞‡±Å‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø
           scanf("%d", &A[i][j]);
       }
   }


   // 3. Matrix B ‡∞á‡∞®‡±ç‡∞™‡±Å‡∞ü‡±ç ‡∞§‡±Ä‡∞∏‡±Å‡∞ï‡±ã‡∞µ‡∞°‡∞Ç
   printf("\nEnter elements of Matrix B:\n");
   for(i = 0; i < rows; i++) {
       for(j = 0; j < cols; j++) {
           scanf("%d", &B[i][j]);
       }
   }


   // 4. ‡∞ï‡±Ç‡∞°‡∞ø‡∞ï ‡∞ö‡±á‡∞Ø‡∞°‡∞Ç (The Logic)
   // ‡∞™‡±ç‡∞∞‡∞§‡∞ø ‡∞¨‡∞æ‡∞ï‡±ç‡∞∏‡±ç ‡∞®‡∞ø ‡∞µ‡∞ø‡∞ú‡∞ø‡∞ü‡±ç ‡∞ö‡±á‡∞∏‡∞ø, ‡∞ï‡±Ç‡∞°‡∞ø, C ‡∞≤‡±ã ‡∞™‡±Ü‡∞°‡±Å‡∞§‡±Å‡∞®‡±ç‡∞®‡∞æ‡∞Ç
   for(i = 0; i < rows; i++) {
       for(j = 0; j < cols; j++) {
           C[i][j] = A[i][j] + B[i][j];
       }
   }


   // 5. ‡∞Ü‡∞®‡±ç‡∞∏‡∞∞‡±ç ‡∞®‡∞ø ‡∞™‡±ç‡∞∞‡∞ø‡∞Ç‡∞ü‡±ç ‡∞ö‡±á‡∞Ø‡∞°‡∞Ç
   printf("\nResultant Matrix C (A + B):\n");
   for(i = 0; i < rows; i++) {
       for(j = 0; j < cols; j++) {
           printf("%d ", C[i][j]);
       }
       // ‡∞í‡∞ï ‡∞∞‡±ã (Row) ‡∞≤‡±ã‡∞®‡∞ø ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç‡∞≤‡±Å ‡∞™‡±ç‡∞∞‡∞ø‡∞Ç‡∞ü‡±ç ‡∞Ö‡∞Ø‡±ç‡∞Ø‡∞æ‡∞ï, ‡∞ï‡±ä‡∞§‡±ç‡∞§ ‡∞≤‡±à‡∞®‡±ç ‡∞ï‡∞ø ‡∞µ‡±Ü‡∞≥‡±ç‡∞≥‡∞æ‡∞≤‡∞ø
       printf("\n");
   }


   return 0;
}
```

## 5\. ‡∞≤‡±à‡∞®‡±ç-‡∞¨‡±à-‡∞≤‡±à‡∞®‡±ç ‡∞µ‡∞ø‡∞µ‡∞∞‡∞£


### ‡∞∏‡±ç‡∞ü‡±Ü‡∞™‡±ç 1: ‡∞°‡∞ø‡∞ï‡±ç‡∞≤‡∞∞‡±á‡∞∑‡∞®‡±ç (Declaration)



‡∞Æ‡±Ü‡∞Æ‡∞∞‡±Ä‡∞≤‡±ã ‡∞Æ‡±Ç‡∞°‡±Å ‡∞ü‡±á‡∞¨‡±Å‡∞≤‡±ç‡∞∏‡±ç ‡∞ï‡±ç‡∞∞‡∞ø‡∞Ø‡±á‡∞ü‡±ç ‡∞ö‡±á‡∞∏‡±ç‡∞§‡±Å‡∞®‡±ç‡∞®‡∞æ‡∞Ç.


 * **A:** ‡∞Æ‡±ä‡∞¶‡∞ü‡∞ø ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞ï‡±ã‡∞∏‡∞Ç.
 * **B:** ‡∞∞‡±Ü‡∞Ç‡∞°‡±ã ‡∞¶‡∞æ‡∞®‡∞ø ‡∞ï‡±ã‡∞∏‡∞Ç.
 * **C:** ‡∞µ‡∞ö‡±ç‡∞ö‡∞ø‡∞® ‡∞Ü‡∞®‡±ç‡∞∏‡∞∞‡±ç ‡∞¶‡∞æ‡∞ö‡±Å‡∞ï‡±ã‡∞µ‡∞°‡∞æ‡∞®‡∞ø‡∞ï‡∞ø.


### ‡∞∏‡±ç‡∞ü‡±Ü‡∞™‡±ç 2: ‡∞á‡∞®‡±ç‡∞™‡±Å‡∞ü‡±ç (‡∞®‡±Ü‡∞∏‡±ç‡∞ü‡±Ü‡∞°‡±ç ‡∞≤‡±Ç‡∞™‡±ç‡∞∏‡±ç ‡∞é‡∞Ç‡∞¶‡±Å‡∞ï‡±Å?)


```c

   for(j = 0; j < cols; j++) {   // Column control
       scanf("%d", &A[i][j]);
   }
}
```


 * `i = 0` (‡∞Æ‡±ä‡∞¶‡∞ü‡∞ø ‡∞∞‡±ã) ‡∞≤‡±ã ‡∞â‡∞®‡±ç‡∞®‡∞™‡±ç‡∞™‡±Å‡∞°‡±Å, ‡∞≤‡±ã‡∞™‡∞≤‡∞ø ‡∞≤‡±Ç‡∞™‡±ç `j = 0, 1, 2...` ‡∞á‡∞≤‡∞æ ‡∞§‡∞ø‡∞∞‡±Å‡∞ó‡±Å‡∞§‡±Å‡∞Ç‡∞¶‡∞ø.
 * ‡∞Ö‡∞Ç‡∞ü‡±á **‡∞Æ‡±ä‡∞¶‡∞ü‡∞ø ‡∞Ö‡∞°‡±ç‡∞°‡±Å ‡∞µ‡∞∞‡±Å‡∞∏ (Row)** ‡∞Æ‡±ä‡∞§‡±ç‡∞§‡∞Ç ‡∞®‡∞ø‡∞Ç‡∞™‡±Å‡∞§‡±Å‡∞Ç‡∞¶‡∞ø.
 * ‡∞Ü ‡∞§‡∞∞‡±ç‡∞µ‡∞æ‡∞§‡±á `i` ‡∞µ‡∞æ‡∞≤‡±ç‡∞Ø‡±Ç `1` ‡∞Ö‡∞µ‡±Å‡∞§‡±Å‡∞Ç‡∞¶‡∞ø, ‡∞Ö‡∞™‡±ç‡∞™‡±Å‡∞°‡±Å **‡∞∞‡±Ü‡∞Ç‡∞°‡±ã ‡∞µ‡∞∞‡±Å‡∞∏** ‡∞®‡∞ø‡∞Ç‡∞™‡±Å‡∞§‡∞æ‡∞Ç.
 * **‡∞ó‡±Å‡∞∞‡±ç‡∞§‡±Å‡∞Ç‡∞ö‡±Å‡∞ï‡±ã‡∞Ç‡∞°‡∞ø:** ‡∞à `i`, `j` ‡∞ï‡∞æ‡∞Ç‡∞¨‡∞ø‡∞®‡±á‡∞∑‡∞®‡±ç ‡∞≤‡±á‡∞ï‡±Å‡∞Ç‡∞°‡∞æ ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞®‡∞ø ‡∞π‡±ç‡∞Ø‡∞æ‡∞Ç‡∞°‡∞ø‡∞≤‡±ç ‡∞ö‡±á‡∞Ø‡∞≤‡±á‡∞Ç.


### ‡∞∏‡±ç‡∞ü‡±Ü‡∞™‡±ç 3: ‡∞ï‡±Ç‡∞°‡∞ø‡∞ï ‡∞≤‡∞æ‡∞ú‡∞ø‡∞ï‡±ç (Addition Logic)


```c
C[i][j] = A[i][j] + B[i][j];
```


‡∞ï‡∞Ç‡∞™‡±ç‡∞Ø‡±Ç‡∞ü‡∞∞‡±ç Matrix A ‡∞≤‡±ã ‡∞í‡∞ï ‡∞ó‡∞°‡∞ø‡∞ï‡∞ø ‡∞µ‡±Ü‡∞≥‡±ç‡∞≤‡∞ø (‡∞â‡∞¶‡∞æ: Row 1, Col 2), Matrix B ‡∞≤‡±ã ‡∞Ö‡∞¶‡±á ‡∞ó‡∞°‡∞ø‡∞≤‡±ã ‡∞â‡∞®‡±ç‡∞® ‡∞®‡∞Ç‡∞¨‡∞∞‡±ç ‡∞®‡∞ø ‡∞§‡±Ä‡∞∏‡±Å‡∞ï‡±Å‡∞®‡∞ø, ‡∞Ü ‡∞∞‡±Ü‡∞Ç‡∞°‡∞ø‡∞ü‡∞ø‡∞®‡∞ø ‡∞ï‡±Ç‡∞°‡∞ø Matrix C ‡∞≤‡±ã ‡∞µ‡±á‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø.


### ‡∞∏‡±ç‡∞ü‡±Ü‡∞™‡±ç 4: ‡∞Ö‡∞µ‡±Å‡∞ü‡±ç-‡∞™‡±Å‡∞ü‡±ç ‡∞™‡±ç‡∞∞‡∞ø‡∞Ç‡∞ü‡∞ø‡∞Ç‡∞ó‡±ç

**‡∞á‡∞¶‡∞ø ‡∞ö‡∞æ‡∞≤‡∞æ ‡∞Æ‡±Å‡∞ñ‡±ç‡∞Ø‡∞Ç.** ‡∞à `printf("\n")` ‡∞¨‡∞Ø‡∞ü‡∞ø ‡∞≤‡±Ç‡∞™‡±ç ‡∞≤‡±ã ‡∞â‡∞Ç‡∞¶‡∞ø, ‡∞ï‡∞æ‡∞®‡±Ä ‡∞≤‡±ã‡∞™‡∞≤‡∞ø ‡∞≤‡±Ç‡∞™‡±ç ‡∞¨‡∞Ø‡∞ü ‡∞â‡∞Ç‡∞¶‡∞ø.

 * ‡∞á‡∞¶‡∞ø ‡∞≤‡±á‡∞ï‡∞™‡±ã‡∞§‡±á, ‡∞Æ‡±Ä ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞ü‡±á‡∞¨‡±Å‡∞≤‡±ç ‡∞≤‡∞æ‡∞ó‡∞æ ‡∞∞‡∞æ‡∞¶‡±Å, ‡∞í‡∞ï‡±á ‡∞™‡±ä‡∞°‡∞µ‡∞æ‡∞ü‡∞ø ‡∞≤‡±à‡∞®‡±ç ‡∞≤‡∞æ‡∞ó‡∞æ ‡∞µ‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø\!

## 6\. ‡∞∏‡∞Æ‡±ç‡∞Æ‡∞∞‡±Ä (Summary)


1. **‡∞Æ‡±ç‡∞Ø‡∞æ‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞Ö‡∞Ç‡∞ü‡±á ‡∞ü‡±á‡∞¨‡±Å‡∞≤‡±ç:** ‡∞°‡±á‡∞ü‡∞æ‡∞®‡∞ø ‡∞Ö‡∞°‡±ç‡∞°‡±Å‡∞ó‡∞æ (Rows), ‡∞®‡∞ø‡∞≤‡±Å‡∞µ‡±Å‡∞ó‡∞æ (Columns) ‡∞∏‡±ç‡∞ü‡±ã‡∞∞‡±ç ‡∞ö‡±á‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø.

     * ‡∞≤‡±ã‡∞™‡∞≤‡∞ø ‡∞≤‡±Ç‡∞™‡±ç (`j`) $\rightarrow$ Columns ‡∞®‡∞ø ‡∞ö‡±Ç‡∞∏‡±Å‡∞ï‡±Å‡∞Ç‡∞ü‡±Å‡∞Ç‡∞¶‡∞ø.
4. **‡∞´‡∞æ‡∞∞‡±ç‡∞Æ‡∞æ‡∞ü‡∞ø‡∞Ç‡∞ó‡±ç:** ‡∞Ö‡∞µ‡±Å‡∞ü‡±ç-‡∞™‡±Å‡∞ü‡±ç ‡∞ü‡±á‡∞¨‡±Å‡∞≤‡±ç ‡∞≤‡∞æ‡∞ó‡∞æ ‡∞∞‡∞æ‡∞µ‡∞æ‡∞≤‡∞Ç‡∞ü‡±á `printf("\n")` ‡∞µ‡∞æ‡∞°‡∞ü‡∞Ç ‡∞Æ‡∞∞‡±ç‡∞ö‡∞ø‡∞™‡±ã‡∞µ‡∞¶‡±ç‡∞¶‡±Å.
# ‡∞∏‡±ç‡∞ü‡±Ç‡∞°‡±Ü‡∞Ç‡∞ü‡±ç‡∞∏‡±ç ‡∞ö‡±á‡∞∏‡±á ‡∞ï‡∞æ‡∞Æ‡∞®‡±ç ‡∞§‡∞™‡±ç‡∞™‡±Å‡∞≤‡±Å

**‡∞é‡∞Ç‡∞¶‡±Å‡∞ï‡±Å:**
