## 1️⃣ రోల్ కాల్ ప్రోగ్రామ్ (Roll Call Program)


మనం క్లాస్ రూమ్ లో అటెండెన్స్ (Attendance) ఎలా తీసుకుంటారో ఉదాహరణగా తీసుకుందాం. ఇది అర్థం చేసుకుంటే `for` loop అర్థం చేసుకోవడం చాలా ఈజీ.


**లాజిక్ సింపుల్:**


1. ఒక్కో రోల్ నంబర్ ని పిలవాలి.
2. ప్రెజెంట్ (1) లేదా ఆబ్సెంట్ (0) అని అడగాలి.
3. చివరగా ఎంతమంది వచ్చారో లెక్క చెప్పాలి.


దానికి సంబంధించిన కోడ్ ఇక్కడ ఉంది:


```c
#include <stdio.h>


int main() {
   int n, i;
   int present = 0, absent = 0;
   int status;


   // క్లాస్ లో మొత్తం ఎంతమంది ఉన్నారు?
   printf("Enter number of students: ");
   scanf("%d", &n);


   // లూప్: ప్రతి స్టూడెంట్ కి అటెండెన్స్ తీసుకోవడం
   for(i = 1; i <= n; i++) {
       printf("Roll number %d (1 = present, 0 = absent): ", i);
       scanf("%d", &status);


       if(status == 1)
           present++;
       else
           absent++;
   }


   printf("\nPresent: %d\n", present);
   printf("Absent : %d\n", absent);


   return 0;
}
```

## 2️⃣ స్యూడో కోడ్ (Pseudocode)


కోడ్ రాయడానికి ముందు, అసలు మనం ఏం చేస్తున్నామో సింపుల్ గా రాసుకుందాం. దాన్నే స్యూడో కోడ్ అంటారు.


**START**


1. మొత్తం స్టూడెంట్స్ ఎంతమంది? (`n` లో తీసుకో).
2. `present` కౌంట్ 0 పెట్టు.
3. `absent` కౌంట్ 0 పెట్టు.
4. **రోల్ నంబర్ 1 నుండి మొదలుపెట్టు.**


**REPEAT (మళ్ళీ మళ్ళీ చెయ్):** చివరి రోల్ నంబర్ (`n`) వచ్చే వరకు:


 * అడుగు: "వచ్చాడా (Present) లేదా రాదా (Absent)?"
 * వస్తే → `present` కౌంట్ పెంచు.
 * రాకపోతే → `absent` కౌంట్ పెంచు.


**చివరగా:**


 * టోటల్ ప్రజెంట్ ఎంతో ప్రింట్ చెయ్.
 * టోటల్ ఆబ్సెంట్ ఎంతో ప్రింట్ చెయ్.
   **END**

## 3️⃣ ఇందులో ఉన్న మూడు ముఖ్యమైన భాగాలు (3 Main Parts)


మనం పైన రాసిన స్టెప్స్ లో, ఆ "REPEAT" అనే పని జరగడానికి **మూడు** విషయాలు చాలా ముఖ్యం:


1. **Initialization (మొదలుపెట్టడం):**
     * ఎక్కడి నుండి స్టార్ట్ చేయాలి?
     * *జవాబు:* రోల్ నంబర్ 1 నుండి (`i = 1`).
2. **Condition (చెక్ చేయడం):**
     * ఎప్పుడు ఆపాలి?
     * *జవాబు:* చివరి స్టూడెంట్ (`n`) అయిపోయాక ఆపాలి (`i <= n`).
3. **Increment (ముందుకు వెళ్ళడం):**
     * నెక్స్ట్ ఎవరిని పిలవాలి?
     * *జవాబు:* తర్వాతి రోల్ నంబర్ కి వెళ్ళాలి (`i++`).


ఈ మూడు (Start, Stop, Move) ఉంటేనే మనం ఏదైనా లెక్కపెట్టగలం.

## 4️⃣ `for` లూప్ ఎందుకు అంత పవర్ఫుల్?


మామూలుగా అయితే ఈ మూడు పనులు వేరు వేరు చోట్ల రాయాలి. కానీ `for` loop లో ఉన్న స్పెషాలిటీ ఏంటంటే—ఈ మూడు పనులని **ఒకే ఒక్క లైన్ లో** రాసేయొచ్చు.


```c
for (i = 1;  i <= n;  i++)
```


చూడండి ఎంత క్లియర్ గా ఉందో:


 * `i = 1` $\rightarrow$ **Start** (ఇక్కడి నుండి మొదలుపెట్టు)
 * `i <= n` $\rightarrow$ **Condition** (ఇక్కడి వరకు చెయ్)
 * `i++` $\rightarrow$ **Increment** (నెక్స్ట్ నంబర్ కి వెళ్ళు)


**అసలు ఇది ఎందుకు కావాలి?**
ఈ లూప్ లేకపోతే, క్లాస్ లో 50 మంది ఉంటే, మీరు 50 సార్లు `printf`, `scanf` రాయాల్సి వస్తుంది. అదే `for` loop వాడితే, లాజిక్ **ఒక్కసలే** రాస్తాం, కానీ కంప్యూటర్ దాన్ని **50 సార్లు** రన్ చేస్తుంది. కోడ్ సింపుల్ గా ఉంటుంది, తప్పులు పోవు\!
