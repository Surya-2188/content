

## ఫిబోనాక్సీ సిరీస్ (Fibonacci Series) అంటే ఏంటి?


ఫిబోనాక్సీ సిరీస్ అనేది నంబర్స్ యొక్క ఒక వరుస. ఇందులో ప్రతి కొత్త నంబర్ రావాలంటే, దాని **ముందున్న రెండు నంబర్లను కూడాలి (Add చేయాలి)**.


ఇది **0, 1** తో మొదలవుతుంది:


 * $0 + 1 = 1$
 * $1 + 1 = 2$
 * $1 + 2 = 3$
 * $2 + 3 = 5$
 * ... ఇలా సాగుతూనే ఉంటుంది.


**సిరీస్:** 0, 1, 1, 2, 3, 5, 8, 13...


కంప్యూటర్ లో దీన్ని ప్రింట్ చేయడానికి మనం లూప్స్ వాడతాం, ఎందుకంటే ప్రతి నంబర్ దాని వెనకాల ఉన్న నంబర్స్ మీద ఆధారపడి ఉంటుంది కాబట్టి.


## 2\. ప్రోగ్రామ్ కాన్సెప్ట్


మనం ఎప్పటిలాగే `#include <stdio.h>` తో మొదలుపెడతాం, ఎందుకంటే `printf`, `scanf` అందులోనే ఉంటాయి. ప్రోగ్రామ్ `main()` లోపల నుండి స్టార్ట్ అవుతుంది.


ఈ సిరీస్ ని స్టోర్ చేయడానికి మనకు మూడు ముఖ్యమైన వేరియబుల్స్ కావాలి:


 * `a = 0`: మొదటి నంబర్.
 * `b = 1`: రెండో నంబర్.
 * `next`: ఈ రెండింటిని కూడితే వచ్చే కొత్త నంబర్.


## 3\. లైన్-బై-లైన్ వివరణ


**Line 3: వేరియబుల్స్ డిక్లేర్ చేయడం**


```c
int n, a = 0, b = 1, next;
```


మనం నాలుగు డబ్బాలు (variables) తీసుకుంటున్నాం:


 * `n`: యూజర్ ఇచ్చే లిమిట్ (ఎక్కడి వరకు ఆపాలి?).
 * `a`, `b`: మొదటి రెండు నంబర్స్ (0, 1 అని ముందే చెప్పేసాం).
 * `next`: కొత్తగా వచ్చే నంబర్ ని ఇందులో దాచుకుంటాం.


**Lines 4-5: ఇన్పుట్ తీసుకోవడం**


```c
printf("Enter the limit: ");
scanf("%d", &n);
```


యూజర్ ని "ఏ నంబర్ వరకు సిరీస్ కావాలి?" అని అడుగుతున్నాం. ఆ వాల్యూ `n` లో స్టోర్ అవుతుంది. `scanf` కి `&` వాడటం మర్చిపోవద్దు\!


**Line 6: మొదట 0, 1 ప్రింట్ చేయడం**


```c
printf("Fibonacci Series: %d %d ", a, b);
```


ఫిబోనాక్సీ ఎప్పుడూ 0, 1 తోనే స్టార్ట్ అవుతుంది కాబట్టి, లూప్ కి ముందే ఆ రెండింటిని ప్రింట్ చేసేస్తున్నాం.


**Line 7: ఫస్ట్ కాలిక్యులేషన్**


```c
next = a + b;
```


తర్వాతి నంబర్ కోసం `a` ని `b` ని కూడుతున్నాం (`0 + 1 = 1`). ఇప్పుడు `next` లో 1 ఉంది.


**Lines 8-13: లూప్ (అసలైన లాజిక్)**


```c
while (next <= n) {
   printf("%d ", next);
   a = b;
   b = next;
   next = a + b;
}
```


యూజర్ ఇచ్చిన లిమిట్ (`n`) దాటేంత వరకు ఈ లూప్ తిరుగుతూనే ఉంటుంది.
లోపల ఏం జరుగుతుందంటే:


1. **ప్రింట్:** `next` లో ఉన్న నంబర్ ని ప్రింట్ చేస్తుంది.
2. **షిఫ్టింగ్ (Shifting):** ఇది చాలా ముఖ్యం. మనం ముందుకు వెళ్ళాలి కాబట్టి:
     * పాత `b` వచ్చి `a` ప్లేస్ లోకి వెళ్తుంది (`a = b`).
     * పాత `next` వచ్చి `b` ప్లేస్ లోకి వెళ్తుంది (`b = next`).
3. **మళ్ళీ కూడటం:** ఇప్పుడు కొత్త `a`, `b` ని కూడి మళ్ళీ `next` ని రెడీ చేస్తుంది.


**Line 14: ముగింపు**


```c
return 0;
```


ప్రోగ్రామ్ సక్సెస్ గా అయిపోయింది అని అర్థం.


## 4\. సమ్మరీ (Summary)


ఈ ప్రోగ్రామ్ ద్వారా మీకు తెలిసేవి:


 * లూప్స్ వాడి ఒక సిరీస్ ని ఎలా తయారు చేయాలి.
 * వేరియబుల్స్ వాల్యూస్ ని ఎలా మారుస్తూ ఉండాలి (`a = b`, `b = next`).
 * ఒక కండిషన్ (`next <= n`) పెట్టి లూప్ ని కరెక్ట్ టైమ్ లో ఎలా ఆపాలి.
